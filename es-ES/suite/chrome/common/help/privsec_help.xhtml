Rastreo de usuarios

Estas opciones le permiten comunicar sus preferencias de rastreo a los sitios web. El rastreo se refiere a que los sitios web (incluyendo anunciantes, proveedores de analíticas y sitios sociales) recogen y analizan datos comprensivos sobre sus patrones de navegación web, afectando por tanto a su privacidad. Para más información sobre este tema, vea ¿Por qué y cómo me están rastreando los sitios web? La tecnología Do Not Track le permite indicar a los sitios web si está de acuerdo en ser rastreado. En esta sección están disponibles las siguientes opciones:

  • Anunciar a los sitios web mis preferencias de rastreo: marque esta casilla para indicar a &brandShortName; que informe a cada sitio web que visite explícitamente (junto con cualquier proveedor de contenido de terceros incrustado en sus páginas) si quiere o no que se rastree sus hábitos de navegación:
    • No quiero ser rastreado: seleccione esta opción para enviar una solicitud Do not track a cada sitio web visitado.
    • Consiento ser rastreado: seleccione esta opción para indicar explícitamente a cada sitio web visitado que está de acuerdo con que se rastreen sus hábitos de navegación.

Notas:

  • Respetar una solicitud Do not track es voluntario, por lo que los sitios web no están obligados a obedecerla. Los sitios y servicios web que siguen la política deberían dejar de recopilar y usar información sobre su patrón de navegación web para toda la actividad para la cual se selecciona Do Not Track.
  • Solicitar Do not track de un sitio web no debería afectar a su capacidad de usar el sitio web correctamente (p.e., al usar una aplicación web o al añadir elementos en un carrito mientras compra).
  • Observe que Do Not Track no impide que se le muestren anuncios mientras navega. Puesto que el sitio web no tiene permitido recopilar y utilizar información sobre usted, el contenido de los anuncios serán menos específicos si no tiene la opción Do Not Track seleccionada.

Navegación segura

Algunos sitios web incluyen contenido potencialmente peligroso y pueden haber sido identificados como maliciosos. Las preferencias localizadas en la parte inferior del panel Privacidad y seguridad le permiten seleccionar qué tipos de sitios identificados de esta forma serán bloqueados. Si se activan &brandShortName; compara la dirección de cada sitio web que vaya a visitar contra listas actualizadas periódicamente. Se emite una advertencia previa a la carga real del sitio identificado, dándole de esta manera la oportunidad de cancelar el proceso de carga. En esta sección están disponibles las siguientes opciones:

  • Bloquear los sitios señalados como atacantes: marque esta casilla si quiere que &brandShortName; le avise si el sitio web que está a punto de visitar ha sido identificado como uno que intenta infectar su equipo con software malicioso. Este tipo de malware puede interferir con las funciones de su equipo o robar su información personal. También es usado frecuentemente para enviar correo spam o para distribuir más malware.
  • Bloquear los sitios señalados como falsificaciones web: marque esta casilla si quiere que &brandShortName; le avise si el sitio web que está a punto de visitar ha sido identificado como un sitio que finge ser un sitio web legítimo de alguna organización o servicio conocidos. Los sitios web como este se crean a menudo para engañarle para que proporcione un nombre de usuario y una contraseña u otra información personal confidencial (conocido normalmente como phishing).

Cuando intenta visitar un sitio web que ha sido identificado como malicioso (y esta característica está activada), verá una de las siguientes advertencias:

  • ¡Página identificada como ataque! Este sitio web ha sido identificado como un sitio web que intenta infectar su equipo con malware.
  • ¡Falsificación web identificada! Este sitio web ha sido identificado como un sitio web que intenta robar tu información personal.

En este punto no se ha producido ningún daño. Seleccione una de las opciones siguientes:

  • ¡Sácame de aquí! No cargar este sitio web. En su lugar, dirigirse a la página de inicio por defecto de &brandShortName;.
  • ¿Por qué se ha bloqueado esta página? Abre una página web que le muestra la información enviada sobre este sitio web concreto (ésta es una página web ubicada en el proveedor de este servicio). Si no existen estos datos del informe, se muestra una página con información general.
  • Ignorar esta advertencia: continuar con la carga de la página. Use esta opción sólo cuando esté seguro de que la página seleccionada es auténtica y visitarla sea seguro. Aparecerá una barra de notificación en la parte superior de la página, dándole la opción de informar al proveedor de este servicio en caso de que se trate de un error:
    • Éste no es un sitio atacante… o Ésta no es una falsificación web…: pulse este botón para abrir una nueva pestaña que le permita informar de advertencias erróneas (ésta es una página externa).

Importante: la característica de navegación segura está activada por defecto. No se recomienda desactivar ninguna de las dos funciones ya que puede dar lugar a visitar sitios no seguros.

Nota: si no se muestra una advertencia para un sitio web, esto no garantiza que un sitio sea confiable; sólo quiere decir que no se ha informado de él. Los sitios web maliciosos pueden parecer seguros pero estar intentando atacarle o robar su información personal en secreto. El propietario del sitio web o su proveedor pueden no saber que el sitio ha sido comprometido.
