Korzystanie z Kuriera Poczty

Kurier Poczty pozwala na wygodne zarządzanie kontaktami internetowymi. W jednym oknie Kuriera możliwe jest ustawienie wielu kont pocztowych i grup dyskusyjnych.

Używanie Kreatora kont pocztowych

Aby utworzyć konto poczty lub grup dyskusyjnych, należy z menu Okno wybrać Kurier Poczty. Jeżeli nie zostały uprzednio utworzone żadne konta pocztowe, podczas pierwszego uruchomienia Kuriera Poczty, zostanie wyświetlone okno Kreatora kont pocztowych.

Kreator kont wyświetli pytania o wszystkie niezbędne informacje, takie jak nazwa użytkownika czy adres e-mail. Jeżeli użytkownik nie posiada wszystkich wymaganych danych, należy przerwać działanie Kreatora i skontaktować się z administratorem sieci lub dostawcą usług internetowych (ISP).

W przypadku, gdy w Kurierze Poczty jest skonfigurowane conajmniej jedno konto Kreator kont pocztowych nie zostanie wyświetlony. W takiej sytuacji Kreatora kont można również uruchomić wybierając Konfiguracja konta z menu Edycja, a następnie naciskając na przycisk Utwórz nowe konto. Więcej w rozdziale Tworzenie dodatkowych kont pocztowych i grup dyskusyjnych.

Konfiguracja kont pocztowych i grup dyskusyjnych

W celu utworzenia nowego konta pocztowego należy podać następujące dane:

  • nazwę użytkownika,
  • adres e-mail,
  • nazwy serwerów poczty przychodzącej oraz wychodzącej,
  • typ serwera poczty przychodzącej (IMAP lub POP)

W celu utworzenia nowego konta grup dyskusyjnych należy podać następujące dane:

  • adres e-mail,
  • nazwę serwera grup dyskusyjnych,
  • nazwę konta.

Aby utworzyć nowe konto pocztowe lub konto grup dyskusyjnych, należy wykonać poniższe czynności, rozpoczynając w oknie Kuriera Poczty:

  1. Z menu Edycja wybrać Konfiguracja kont. Wyświetlone zostanie okno dialogowe Konfiguracja kont.
  2. Nacisnąć przycisk Dodaj konto.

    Wymagane przez Kreatora dane zależą od typu konta wybranego w jego pierwszym oknie. Czcionką pogrubioną zaznaczono pola, które wyświetlane są podczas konfiguracji konta pocztowego.

  3. Konfiguracja konta: Należy wybrać rodzaj konfigurowanego konta, a następnie nacisnąć przycisk Dalej.
  4. Tożsamość: Należy wprowadzić nazwę użytkownika oraz adres e-mail, a następnie nacisnąć przycisk Dalej.
  5. Informacje o serwerze:
    • Należy określić typ serwera poczty przychodzącej (POP lub IMAP). Nie wszyscy dostawcy internetowi obsługują oba standardy. Więcej informacji w rozdziale Ustawienia kont pocztowych i grup dyskusyjnych — Konfiguracja serwerów.
    • Wprowadzić nazwę serwera poczty przychodzącej.
    • Jeżeli użytkownik chce, aby to konto było częścią Globalnego skrzynki odbiorczej, to opcja Użyj globalnej skrzynki odbiorczej powinna być zaznaczona. Wiadomości będą zachowywane w Folderach lokalnych. W przeciwnym przypadku wiadomości będą zachowywane we własnym folderze konta.

      Uwaga: Domyślnie Kurier Poczty będzie używał jednego serwera poczty wychodzącej (SMTP), nawet jeżeli użytkownik używa różnych kont pocztowych. Nazwa serwera SMTP może nie być bezpośrednio wyróżniona w informacjach o ustawieniach konta dostarczonych użytkownikowi. Przykładowo, serwer SMTP może być tym samym komputerem, co serwer POP lub IMAP. W razie wątpliwości należy skontaktować się z dostawcą usługi poczty elektronicznej lub administratora systemu.

    • Nacisnąć przycisk Dalej.
  6. Nazwa użytkownika: Należy wprowadzić nazwę użytkownika na serwerze podaną przez dostawcę usługi poczty elektronicznej, a następnie nacisnąć przycisk Dalej.
  7. Nazwa konta: Należy wprowadzić dowolną nazwę, która pozwoli zidentyfikować tworzone konto. Aby przejść do następnej planszy, należy nacisnąć przycisk Dalej.
  8. Gratulacje! Należy sprawdzić czy podane wcześniej informacje są poprawne. W razie wątpliwości należy skontaktować się dostawcą usługi poczty elektronicznej lub usług internetowych (ISP). Aby zakończyć działanie kreatora i utworzyć nowe konto, należy nacisnąć przycisk Zakończ.
  9. Nowoutworzone konto pojawi się na liście w lewej części okna Konfiguracja kont. Aby rozpocząć używanie nowego konta, należy zamknąć okno konfiguracji kont przyciskając OK.

Utworzone konto jest gotowe do odbierania wiadomości. Kurier Poczty &brandShortName; będzie prosił użytkownika o hasło na serwerze przy pierwszym połączeniu z serwerem w każdej sesji. Szczegółowy opis odbierania wiadomości w rozdziale Pobieranie wiadomości.

Tworzenie dodatkowych kont pocztowych i grup dyskusyjnych

Okno Konfiguracja konta umożliwia rozpoczęcie tworzenia nowego konta, a także zmianę takich ustawień istniejących kont, jak:

  • konfiguracja serwera pocztowego lub serwera grup dyskusyjnych (na przykład opcje połączenia lub pobierania wiadomości),
  • przechowywanie kopii wiadomości (ustawienia folderów),
  • adres zwrotny, nazwa instytucji oraz sygnaturka.

W celu utworzenia nowego konta lub zmiany ustawień istniejącego konta, należy:

  1. Z menu Edycja wybrać Konfiguracja konta. Pojawi się okno Konfiguracja konta, gdzie dostępne są opcje:
    • Dodaj konto: Należy nacisnąć ten przycisk, aby dodać nowe konto pocztowe lub grup dyskusyjnych. W pokazanych oknach kreatora należy wpisać dokładnie podane informacje dotyczące ustawień konta/serwerów. Korzystając z przycisków Dalej i Wstecz można się przemieszczać pomiędzy ekranami kreatora. Aby anulować działanie kreatora, należy nacisnąć przycisk Anuluj.
    • Ustaw jako domyślne: Ustawia zaznaczone konto jako domyślne. Konto domyślne jest wyświetlane na początku listy kont w oknie Kuriera Poczty. Dokonane zmiany zostaną zastosowane po ponownym otwarciu okna Kuriera Poczty.

      Kurier Poczty będzie automatycznie sprawdzał, czy na koncie domyślnym znajdują się nowe wiadomości. (W przypadku, gdy serwer poczty przychodzącej domyślnego konta jest serwerem typu POP, Kurier Poczty nie będzie pobierał wiadomości, dopóki nie zostanie naciśnięty przycisk Pobierz na pasku narzędzi w głównym oknie programu.)

    • Usuń konto: Naciśnięcie tego przycisku spowoduje usunięcie konta, wszystkich jego wiadomości i ustawień.
    • Serwer poczty wychodzącej (SMTP): Opcja (znajdująca się na końcu listy kont) pozwala zmodyfikować dane serwerów poczty wychodzącej. Więcej informacji w rozdziale Ustawienia Kuriera Poczty — Serwer poczty wychodzącej (SMTP).
  2. Zaznaczając nagłówki pod nazwą konta na liście po lewej, można modyfikować ustawienia w panelu po prawej stronie okna.
  3. Aby zachować zmiany, należy nacisnąć przycisk OK.

Zmiana ustawień istniejącego konta

Aby wyświetlić lub zmienić dane dotyczące istniejącego konta pocztowego lub grupy dyskusyjnej, należy rozpoczynając w oknie Kuriera Poczty:

  1. Z menu Edycja wybrać Konfiguracja konta.
  2. Wybrać nazwę konta po lewej stronie okna Konfiguracji kont. W prawej części okna zostaną wyświetlone informacje o wybranym koncie, takie jak adres e-mail lub nazwa pliku sygnaturki.
  3. Aby wyświetlić pozostałe opcje, należy kliknąć w żądane pozycje pod nazwą konta:
    • Konfiguracja serwera: Opcja pozwala na zmianę ustawień serwera pocztowego lub grup dyskusyjnych. Dostępne opcje zależą od rodzaju wybranego serwera (IMAP, POP3 lub serwer grup dyskusyjnych). Więcej informacji w rozdziale Ustawienia kont pocztowych i grup dyskusyjnych — Konfiguracja serwerów.

      Ważne: Jeżeli konieczna jest zmiana typu serwera (IMAP, POP lub grup dyskusyjnych) konieczne jest usunięcie istniejącego konta. Następnie należy zamknąć i ponownie uruchomić &brandShortName;, a następnie ponownie utworzyć konto, używając nowych ustawień.

    • Kopie i foldery: Zawiera ustawienia określające zakres tworzenia automatycznych kopii wysyłanych wiadomości (folder wiadomości wysłanych, niewidoczna kopia blind carbon copy) oraz w jakich folderach będą przechowywane szkice i szablony. Więcej informacji w rozdziale Ustawienia kont pocztowych i grup dyskusyjnych — Kopie i foldery.
    • Komponowanie: Pozwala zmienić ustawienia domyślnego formatu oraz cytowania w wysyłanych wiadomościach. Można także nadpisać globalne ustawienia serwerów katalogowych LDAP dla wszystkich książek adresowych w oknie dialogowym Preferencji. Więcej informacji w rozdziale Ustawienia kont pocztowych i grup dyskusyjnych — Komponowanie.
    • Sieć i miejsce na dysku: Ustawienia te dotyczą pracy w trybie offline (bez połączenia z siecią Internetem) i pozwalają ograniczyć czas połączenia i miejsce na dysku zajęte przez pobierane wiadomości. Więcej informacji w rozdziale Synchronizacja i miejsce na dysku dla konta IMAP lub Synchronizacja i miejsce na dysku dla konta grup dyskusyjnych.
    • Miejsce na dysku (konta POP): Pozwala ustawić maksymalny rozmiar wiadomości pobieranych z serwera na dysk komputera użytkownika. Więcej informacji w rozdziale Ustawienia miejsca na dysku dla konta POP.
    • Zabezpieczenia: Pozwala ustawić które certyfikaty będą używane do podpisywania i szyfrowania wysyłanych wiadomości. Podpisy cyfrowe pozwalają na identyfikację nadawcy wiadomości. Szyfrowanie pozwala zachować poufność przesyłanych informacji. Więcej informacji w rozdziale Ustawienia kont pocztowych i grup dyskusyjnych — Zabezpieczenia.
  4. Aby zachować wprowadzone zmiany, należy nacisnąć przycisk OK.

Pobieranie nowych wiadomości

Jeżeli domyślnym kontem jest konto na serwerze IMAP, podczas uruchamiania Kurier Poczty automatycznie pobierze nowe wiadomości i wyświetli je w folderze Odebrane.

W przypadku kont pocztowych, które są obsługiwane przez serwer POP, należy nacisnąć przycisk Pobierz, aby pobrać nowe wiadomości. Domyślnie wiadomości z konta POP są usuwane z serwera po pobraniu. Zachowanie to można zmienić w ustawieniach serwera POP.

Możliwe jest ustawienie Kuriera Poczty tak, aby automatycznie pobierał nowe wiadomości po uruchomieniu, a następnie po upływie określonego czasu sprawdzał, czy są nowe wiadomości.

Powiadomienie o nowej poczcie

Aby zmienić ustawienia Kuriera Poczty dotyczące automatycznego pobierania nowych wiadomości, należy, rozpoczynając w oknie Kuriera Poczty:

  1. Z menu Edycja wybrać Konfiguracja konta.
  2. W przypadku posiadania wielu kont należy wybrać konto, którego ustawienia mają zostać zmienione, a następnie wybrać kategorię Ustawienia serwera.
  3. Zaznaczyć jedną lub obie poniższe opcje:
    • Sprawdzaj podczas uruchamiania, czy są nowe wiadomości: Należy zaznaczyć tę opcję, aby automatycznie sprawdzać stan skrzynki pocztowej podczas uruchamiania Kuriera Poczty. Aby automatycznie pobrać nowe wiadomości z konta typu POP, należy również zaznaczyć opcję Automatycznie pobieraj nowe wiadomości.
    • Sprawdzaj czy nadeszły nowe wiadomości co [__] minut: Należy zaznaczyć tę opcję, aby Kurier Poczty, po upływie określonego czasu, automatycznie sprawdzał, czy na serwerze są nowe wiadomości. Niezależnie od ustawień tej opcji, pobranie nowych wiadomości będzie zawsze możliwe, po naciśnięciu przycisku Pobierz w głównym oknie Kuriera Poczty.
  4. Nacisnąć klawisz OK. Wprowadzone zmiany zostaną uaktywnione po ponownym uruchomieniu Kuriera Poczty.

W rozdziale Preferencje Kuriera Poczty — Powiadamianie opisano jak ustawić Kuriera Poczty, aby odgrywał dźwięk po odebraniu nowych wiadomości.

Aby pobrać nowe wiadomości, należy:

  • Nacisnąć przycisk Pobierz w głównym oknie Kuriera Poczty.
  • Z menu Plik wybrać opcję Pobierz nowe wiadomości.

Aby pobrać nowe wiadomości dla wszystkich kont, należy:

  1. Nacisnąć trójkąt na przycisku Pobierz na pasku narzędzi.
  2. Wybrać pozycję Pobierz wszystkie nowe wiadomości z wyświetlonego menu.

    W przypadku, gdy użytkownik nie jest zalogowany do kont pocztowych, Kurier Poczty przed pobraniem poczty wyświetli okno dialogowe z pytaniem o nazwę użytkownika i hasło dla danego konta. (Wszystkie hasła mogą być przechowywane przez Menedżera Haseł i wprowadzane automatycznie w razie potrzeby.)

Uwaga: Można także skorzystać z opcji Pobierz nowe wiadomości z z menu Plik.

Aby pobrać nowe wiadomości z wybranego konta, należy, rozpoczynając w oknie Kuriera Poczty:

  1. Nacisnąć ikonkę trójkąta na przycisku Pobierz na pasku narzędzi Kuriera Poczty.
  2. Wybrać z listy konto, z którego Kurier Poczty ma pobrać nowe wiadomości.

Uwaga: Podczas pierwszego pobierania wiadomości z konta Kurier Poczty wyświetli pytanie o hasło, jakiego należy użyć do autoryzacji połączenia z serwerem. Można wtedy zaznaczyć opcję Użyj Menedżera Haseł, aby zapamiętać to hasło, aby hasło dla tego konta było wprowadzane automatycznie.

Zapamiętane nazwy użytkowników i hasła przechowywane są przez Menedżera Haseł w pliku, którego odczytanie jest trudne, jednakże nie jest niemożliwe. Hasła zapamiętane przez Menedżera Haseł można szyfrować lub usunąć, lub w inny sposób modyfikować działanie menedżera. Więcej informacji w rozdziale Używanie Menedżera haseł.

Zmiana wyglądu głównego okna Kuriera Poczty

Możliwa jest zmiana następujących ustawień wpływających na wygląd głównego okna Kuriera Poczty (okna, które jest otwierane powybraniu Kuriera Poczty w menu Okno):

  • Rozwinięcie lub zwinięcie dowolnego panelu za pomocą uchwytu umożliwia przełączanie pomiędzy widokiem układu dwóch lub trzech paneli.

Sortowanie oraz grupowanie wiadomości w wątki

Aby sortować wiadomości według takich kategorii jak temat, nadawca, data lub priorytet wiadomości, należy:

  • Nacisnąć odpowiedni nagłówek kolumny w głównym oknie Kuriera Poczty. Można również wybrać Sortuj z menu Widok i wskazać kategorię, według której wiadomości mają być sortowane.

Aby zmienić kolejność nagłówków kolumn, należy:

  • Przeciągnąć myszą kolumnę w prawo lub lewo, w żądane miejsce.

Aby zmienić listę wyświetlanych kolumn, należy:

  • Nacisnąć ikonę wyboru kolumn i wybrać kolumnę, która ma być dodana do listy lub z niej usunięta.

Aby sortować wiadomości według wątku (tematu), tak aby każda wiadomość została zgrupowana razem z odpowiedziami na nią, należy:

  • Nacisnąć przycisk Wątek (przycisk stanowi zarazem pierwszy nagłówek, poprzedzający kolumny Temat, Nadawca i Data) lub wybrać Sortuj z menu Widok i wskazać kategorię Wątek.
Przycisk Wątek

Podpowiedź: Należy wybrać opcję Zachowuj wątki podczas sortowania w preferencjach Kuriera Poczty jeżeli &brandShortName; ma zachowywać grupowanie w wątki podczas sortowania wiadomości przez naciskanie nagłówków kolumn. Przycisk Wątek w tym trybie zmienia tylko wyświetlanie listy wiadomości na pogrupowane lub niepogrupowane w wątki. jeżeli opcja Zachowuj wątki podczas sortowania nie jest wybrana, to &brandShortName; automatycznie wyświetla wiadomości jako niepogrupowane po posortowaniu ich przez naciskanie nagłówków kolumn.

Podpowiedź: Aby ułatwić odnalezienie nieprzeczytanych wiadomości w zwiniętych wątkach, Kurier Poczty podkreśla pierwszą wiadomość w wątku.

Zapisywanie oraz drukowanie wiadomości

Aby zapisać wiadomość w formacie tekstowym lub pliku HTML, należy:

  1. Wybrać wiadomość w oknie Kuriera Poczty.
  2. Z menu Plik wybrać Zapisz jako i następnie Plik.
  3. Wybrać typ pliku (Pliki HTML, Pliki tekstowe, lub Pliki poczty). Typ Plik poczty pozwala zapisać wiadomość w formacie odczytywanym przez program Microsoft Outlook lub Outlook Express.
  4. Zmienić rozszerzenie pliku na .html, .txt, lub .eml, w zależności od typu wybranego w kroku 3.
  5. Wybrać folder docelowy i nacisnąć przycisk Zapisz.

Aby wydrukować wybraną wiadomość, należy:

  • Kliknąć przycisk Drukuj na pasku narzędzi lub z menu Plik wybrać Drukuj.

Kontrola obrazków, skryptów i wtyczek

Domyślnie Kurier Poczty nie wyświetla w otrzymanych wiadomościach obrazków i innej treści ze zdalnych serwerów, z wyjątkiem wiadomości od nadawców z książki adresowej użytkownika, dla których zostało to dozwolone. Aby zmienić te ustawienia, należy:

  1. Z menu Edycja &brandShortName; wybrać Preferencje.
  2. W kategorii Kurier Poczty wybrać Wiadomości. (Jeśli podkategorie nie są widoczne, to aby rozwinąć listę, należy podwójnie kliknąć na Kurier Poczty.)
  3. Odznaczyć opcję Blokuj obrazki i inną treść ze zdalnych serwerów.
  4. Nacisnąć przycisk OK, aby zatwierdzić zmiany.

Uwaga: Przejrzyj informacje o opcji Zezwól na zdalne obrazki w wiadomościach HTML w rozdziale Tworzenie nowej wizytówki w książce adresowej, aby uzyskać więcej informacji o tym jak określić, w wiadomościach od których nadawców będą wyświetlane treści ze zdalnych serwerów.

W ustawieniach domyślnych Kurier Poczty nie wykonuje skryptów JavaScript w wiadomościach pocztowych oraz wtyczki przeglądarki są wyłączone. Aby zmienić te ustawienia, należy:

  1. Z menu Edycja &brandShortName; wybrać Preferencje.
  2. W kategorii Zaawansowane wybrać Skrypty i wtyczki. (Jeśli podkategorie nie są widoczne, to aby rozwinąć listę, należy podwójnie kliknąć na Zaawansowane.)
    • W polu Włącz interpretację kodu JavaScript w komponentach zaznaczyć opcję Kurier Poczty, aby włączyć obsługę języka JavaScript w wiadomościach.
    • Aby wyłączyć obsługę wtyczek w Kurierze Poczty, należy odznaczyć opcję Kurier Poczty w polu Włącz obsługę wtyczek w komponencie.
  3. Nacisnąć przycisk OK, aby zatwierdzić zmiany.

Tworzenie nowych wiadomości pocztowych i grup dyskusyjnych

Aby napisać, zaadresować i wysłać nową wiadomość lub odpowiedzieć na istniejącą, używając okna tworzenia wiadomości, należy wykonać jedną z poniższych czynności:

  • W oknie przeglądarki lub Kompozytora Stron z menu Plik wybrać Nowy i wskazać Wiadomość.
  • W oknie Kuriera Poczty nacisnąć przycisk Nowa na pasku narzędzi.
  • Jeżeli w bieżącym oknie wyświetlona jest wiadomość, nacisnąć jeden z przycisków na pasku narzędzi: Odpowiedz, Odpowiedz wszystkim lub Przekaż.
  • W oknie Książki adresowej wybrać adresata z listy adresów i nacisnąć przycisk Nowa wiadomość na pasku narzędzi.

Podpowiedź: by użyć wbudowanego edytora HTML podczas tworzenia wiadomości wysyłanych z danego konta, należy zaznaczyć opcję Podczas tworzenia wiadomości używaj edytora HTML we właściwościach danego konta. (Można określić inny edytor dla poszczególnych kont.) Więcej informacji w rodziale Zmiana ustawień istniejącego konta.

Wiadomości w formacie HTML mogą zawierać różne kroje pisma, kolory, listy numerowane i wypunktowane. Należy jednak pamiętać, że niektórzy odbiorcy mogą nie być w stanie przeczytać takich wiadomości. Wysyłanie wiadomości grup dyskusyjnych w formacie HTML jest również niemile widziane. Aby utworzyć wiadomość w formacie tekstowym (mając ustawiony domyślny format HTML), należy nacisnąć przycisk Nowa lub Odpowiedz na pasku narzędzi trzymając jednocześnie wciśnięty klawisz Shift. Aby utworzyć wiadomosć w formacie HTML (mając ustawiony domyślny format tekstowy), należy nacisnąć przycisk Nowa lub Odpowiedz na pasku narzędzi trzymając jednocześnie wciśnięty klawisz Shift.

Korzystanie z okna tworzenia wiadomości

Aby napisać, zaadresować i wysłać wiadomość pocztową lub wiadomość grup dyskusyjnych, należy użyć okna tworzenia wiadomości. Dla każdego konta możliwe jest określenie domyślnego edytora (tekstowego lub HTML), który będzie używany przy tworzeniu wiadomości. Domyślny edytor tekstowy może zostać zmieniony w kategorii Komponowanie okna konfiguracji konta (z menu Edycja Kuriera Poczty należy wybrać pozycję Konfiguracja kont).

Aby otworzyć okno tworzenia wiadomości, należy nacisnąć przycisk Nowa na pasku narzędzi Kuriera Poczty.

Okno tworzenia wiadomości zawiera następujące elementy:

  • Pasek narzędzi

    Pasek zawiera następujące przyciski:

    • Wyślij: Wysyła wiadomość.
    • Adres: Wyświetla adresy zebrane w książkach adresowych.
    • Dołącz: Załącza pliki do wiadomości. Więcej informacji w rozdziale Załączniki.
    • Pisownia: Sprawdza poprawność pisowni wiadomości.
    • Zabezpieczenia: Wyświetla informacje o szyfrowaniu i/lub podpisie cyfrowym dla wiadomości.
    • Zapisz: Zapisuje wiadomość do późniejszej edycji (w folderze Szkice).
  • Pole adresów: pozwala wpisać adresy odbiorców wiadomości.
  • Pole załączników: Nazwy wszystkich załączonych plików do wiadomości są wyświetlone w polu załączników, po prawej stronie pola adresów.
  • Pole wiadomości: pozwala wpisać treść wiadomości.

Jeżeli do tworzenia wiadomości wybrano edytor HTML, to wyświetlony zostanie dodatkowy pasek formatowania tekstu podobny do paska używanego w Kompozytorze Stron.

W celu uzyskania bliższych informacji na temat tworzenia wiadomości w formacie HTML, należy kliknąć na odnośnik Formatowanie stron.

Adresowanie wiadomości

Aby zaadresować wiadomość, należy:

  1. Wprowadzić nazwę odbiorcy w polu adresu.

    Jeżeli funkcja autouzupełniania adresów jest włączona (domyślnie jest włączona), wystarczy wprowadzić kilka pierwszych liter nazwy adresata i poczekać, aż Kurier Poczty uzupełni adres. (Można również wprowadzić cześć nazwy adresata i nacisnąć ReturnEnter, aby użyć pierwszego pasującego adresu.)

  2. Jeżeli do wprowadzonego wzoru pasuje więcej niż jeden adres, należy wybrać adres z listy i nacisnąć klawisz ReturnEnter.

    Uwaga: Aby wprowadzić wiele adresów w jednej linii, należy użyć przecinka do oddzielenia adresów od siebie. Nie należy używać przecinka do oddzielenia wyświetlanej nazwy użytkownika. Na przykład:

    user1@netscape.net, user2@netscape.net

  3. Jeśli wiadomość ma być wysłana z innego konta, niż aktualnie ustawione, należy wybrać żądane konto z menu rozwijalnego Nadawca. Więcej informacji w rozdziale Zmiana konta, z którego zostanie wysłana wiadomość.
  4. Jeżeli zachodzi konieczność zmiany typu odbiorcy, należy kliknąć na polu Do: i wybrać jeden z poniższych typów:
    • Do: Określa podstawowych odbiorców wiadomości.
    • Cc: Określa dodatkowych odbiorców wiadomości (odbiorcy otrzymają kopię wiadomości Cc:).
    • Ukryta kopia: Określa dodatkowych odbiorców wiadomości, których tożsamość będzie ukryta przed innymi odbiorcami, włączając w to adresatów określonych w polu Kopia: (odbiorcy otrzymają ukrytą kopię wiadomości Bcc:).
    • Odpowiedź do: Określa adres zwrotny, inny niż adres z którego wysyłana jest wiadomość.
    • Grupa dyskusyjna: Określa adres grupy dyskusyjnej, do której zostanie przesłana wiadomość.
    • Inne grupy: Określa adres innej grupy dyskusyjnej, do której, zamiast do grupy podstawowej, zostaną przekierowane wszystkie odpowiedzi na wysyłaną wiadomość.

Podpowiedź: by użyć szybkiego adresowania wiadomości, należy kliknąć prawym klawiszem myszki na adresie e-mail zawartym w wyświetlanej wiadomości, a następnie wybrać z menu pozycję Wyślij wiadomość do.

Zmiana konta, z którego zostanie wysłana wiadomość

Jeżeli skonfigurowano więcej niż jedno konto pocztowe, wiadomość wysyłana jest z konta (lub serwera), które było aktywne w czasie rozpoczęcia tworzenia wiadomości. Kurier Poczty umożliwia zmianę domyślnego nadawcy wiadomości po rozpoczęciu tworzenia wiadomości. Aby zmienić nadawcę wiadomości, należy nacisnąć pole Nadawca i wybrać nowego nadawcę z listy. Kopia wiadomości zostanie zachowana w folderze Wiadomości wysłane przypisanym do konta, które zostało użyte do wysłania wiadomości

Autouzupełnianie adresów

Funkcja autouzupełniania adresów ułatwia wprowadzanie adresów podczas tworzenia wiadomości, eliminując konieczność wyszukiwania adresów lub wprowadzania adresów w kompletnej formie. Podczas wprowadzania pierwszych liter adresu, Kurier Poczty wyszukuje i wyświetla listę wszystkich pasujących adresów znajdujących się w książkach adresowych oraz na serwerze usług katalogowych LDAP (jeżeli jest dostępny). Zapobiega to pomyłkom w trakcie wprowadzania adresu. Jeżeli zostanie odnaleziony tylko jeden adres pasujący do wprowadzanej nazwy, Kurier Poczty wyświetli go w polu adresowym. Funkcja autouzupełniania adresów domyślnie jest włączona.

W przypadku, gdy użytkownik nie chce skorzystać z podpowiedzi Kuriera Poczty, należy nacisnąć klawisz Backspace lub Delete, usunąć podpowiedź i wpisać inny adres.

Aby całkowicie zablokować autouzupełnianie adresów, należy:

  1. Z menu Edycja &brandShortName; wybrać Preferencje.
  2. W kategorii Kurier Poczty wybrać Adresowanie. (Jeśli podkategorie nie są widoczne, to aby rozwinąć listę, należy podwójnie kliknąć na Kurier Poczty.)
  3. W polu Autouzupełnianie adresów należy odznaczyć opcje wyszukiwania adresów W lokalnych książkach adresowych oraz Na serwerze usług katalogowych.
  4. Nacisnąć przycisk OK.

Opcje wysyłania wiadomości

Podczas tworzenia wiadomości możliwy jest dostęp do poniższych funkcji przez menu Opcje:

  • Wybierz adresy: Umożliwia wybór adresu odbiorcy wiadomości z dostępnych Książek adresowych. W wyświetlonym oknie należy wybrać adres, a następnie nacisnąć na przycisk Do:, Kopia: lub Ukryta kopia:, aby dodać adresata do listy odbiorców wiadomości.
  • Sprawdź pisownię: Sprawdza pisownię wiadomości. Ta sama czynność zostanie wykonana po naciśnięciu przycisku Pisownia na pasku narzędzi.
  • Sprawdzaj pisownię w trakcie tworzenia wiadomości: Pisownia jest sprawdzana podczas wprowadzania tekstu tworzonej wiadomości.
  • Żądaj potwierdzenia dostarczenia wiadomości: Należy zaznaczyć tę opcję, aby uzyskać potwierdzenie otwarcia wiadomości. Adresat może jednak nie wysłać potwierdzenia. Opcja ta pozwala na przełączanie żądania potwierdzenia dla pojedynczych wiadomości. Aby Kurier Poczty automatycznie żądał potwierdzeń, należy ustawić preferencje potwierdzeń odbioru. Więcej informacji w rozdziale Preferencje Kuriera Poczty — Potwierdzenia.
  • Format: Pozwala zmienić format w jakim zostanie wysłana wiadomość. Możliwe opcje to format tekstowy, HTML lub oba formaty. W przypadku wyboru opcji Wybierz automatycznie, Kurier Poczty dobierze automatycznie format wysyłanych wiadomości lub wyświetli pytanie o format, jeżeli nie jest wiadomo, czy odbiorca może otwierać wiadomości HTML. Wybrane ustawienia tymczasowo zmieniają domyślne ustawienia formatu, określone w kategorii Format w Preferencjach.
  • Priorytet: Pozwala zmienić priorytet wiadomości na: najniższy, niski, normalny, wysoki lub najwyższy.
  • Kodowanie znakow: Umożliwia wybór kodowania znaków dla tej wiadomości.
  • Archiwizuj kopię wysyłanej wiadomości: Umożliwia wybór folderu, w którym zostanie zachowana dodatkowa kopia wiadomości, niezależnie od ustawień dotyczących domyślnego tworzenia kopii wysyłanych wiadomości.
  • Zabezpieczenia wiadomości: Umożliwia zmianę domyślnych opcji zabezpieczeń dla tworzonej wiadomości.

Dodatkowa opcja formatowania wiadomości jest dostępna w menu Edycja:

  • Zawiń ponownie: Jeżeli tworzona podczas edycji w trybie tekstowym wiadomość zawiera długie linie, można użyć funkcji Zawiń ponownie, aby zawinąć tekst wiadomości. Polecenie to zawija wiersze dłuższe od liczby znaków ustawionej w preferencjach Komponowania. Jest to szczególnie przydatne w przypadku odpowiadania na wiadomość, która zawiera długie linie, nie mieszczące się w oknie komponowania wiadomości. Możliwe jest zawijanie linii tylko w zaznaczonej części wiadomości lub w całej wiadomości, jeżeli nie wybrano zaznaczenia.

    Po otworzeniu okna Konfiguracja konta opcją z menu Edycja, możliwe jest określenie domyślnego edytora używanego podczas tworzenia wiadomości. Aby użyć wbudowanego edytora HTML, należy zaznaczyć opcję Podczas tworzenia wiadomości używaj edytora HTML. Jeżeli opcja ta nie jest zaznaczona, wiadomości będą edytowane w trybie tekstowym. Aby tymczasowo przełączyć domyślnie wybrany edytor wiadomości, należy przytrzymać Shift na klawiaturze podczas naciskania na przycisk Nowa lub Odpowiedz na pasku narzędzi Kuriera Poczty.

Tworzenie odpowiedzi na wiadomość

Aby odpowiedzieć na wiadomość, należy:

  • Wybrać wiadomość.
  • Nacisnąć przycisk Odpowiedz, aby wysłać odpowiedź wyłącznie do nadawcy wiadomości.
  • Nacisnąć przycisk Odpowiedz wszystkim, aby wysłać odpowiedź do wszystkich adresatów wiadomości, na którą się odpowiada.

Aby treść oryginalnej wiadomości była domyślnie dołączana do każdej tworzonej odpowiedzi, należy:

  1. Z menu Edycja wybrać Konfiguracja kont.
  2. W przypadku, gdy użytkownik posiada kilka kont, wybrać konto i otworzyć kategorię Komponowanie.
  3. Zaznaczyć opcję Automatycznie cytuj oryginalną wiadomość podczas tworzenia odpowiedzi.
  4. Określić, w którym miejscu wiadomości rozpocząć odpowiedź.
  5. Gdy użytkownik korzysta z sygnaturki (podpisu), można także zdefiniować gdzie ma być umieszczona:
    • Pod cytowanym tekstem (zalecane) — na końcu wiadomości pod cytowanym tekstem.
    • Poniżej odpowiedzi (ponad cytowanym tekstem) — pomiędzy odpowiedzią i cytowanym tekstem.

    Uwaga: Jeżeli użytkownik ma utworzoną sygnaturkę, to może ją pominąć w czasie odpowiadania na wiadomość.

  6. Nacisnąć przycisk OK.

Przesyłanie wiadomości

Podczas przesyłania wiadomości do innych adresatów można określić w jaki sposób zostanie przesłana zawartość oryginalnej wiadomości. Dostępne opcje to: bezpośrednio (jako treść tworzonej wiadomości, sposób domyślny) lub jako załącznik.

Aby przesłać wiadomość, należy:

  1. Wybrać wiadomość, a następnie nacisnąć przycisk Prześlij.
  2. Wprowadzić adres odbiorcy wiadomości.
  3. Nacisnąć przycisk Wyślij na pasku narzędzi.

Aby zmienić domyślne ustawienia przesyłania wiadomości, należy:

  1. Z menu Edycja &brandShortName; wybrać Preferencje.
  2. W kategorii Kurier Poczty wybrać Tworzenie. (Jeśli podkategorie nie są widoczne, to aby rozwinąć listę, należy podwójnie kliknąć na Kurier Poczty.)
  3. Wybrać żądany sposób przekazywania wiadomości: bezpośrednio (jako treść tworzonej wiadomości) lub jako załącznik.

    Uwaga: Jeżeli użytkownik ma utworzoną sygnaturkę i przesyła wiadomość w treści wiadomości, to jej umieszczenie jest zależne od odpowiedniego ustawienia odpowiadania na wiadomość. Opcjonalnie można pominąć sygnaturkę przy przesyłaniu wiadomości dalej.

  4. Nacisnąć przycisk OK.

Podopowiedź: Aby wybrać sposób przesłania konkretnej wiadomości, należy wybrać wiadomość, a następnie z menu Wiadomość wybrać Przekaż wiadomość i dalej Bezpośrednio lub Jako załącznik.

Potwierdzenia otwarcia (dostarczenia) wiadomości

Można uzyskać potwierdzenie dostarczenia wiadomości przez odbiorcę. Konieczne jest aby odbiorca używał programu pocztowego obsługującego standard Message Disposition Notification (MDN). Należy jednak pamiętać, że odbiorca może zdecydować nie odsyłać potwierdzenia. Dla wiadomości wysłanych do grup dyskusyjnych nie można uzyskać potwierdzenia ponieważ serwery News nie obsługują tej funkcji.

Aby zażądać potwierdzenia dostarczenia dla wyszystkich wysyłanych wiadomości, należy użyć globalnych preferencji Potwierdzenia, które pozwalają określić zasady obsługi żądań potwierdzeń. Ustawienia te mogą być także zmienione dla poszczególnych kont.

Aby zarządać potwierdzenia otwarcia dla pojedynczej wiadomości, należy:

  • W oknie tworzenia wiadomości z menu Opcje wybrać Żądaj potwierdzenia dostarczenia wiadomości.

Aby automatycznie żądać potwierdzenia dostarczenia wiadomości dla każdej wiadomości wysyłanej z każdego konta, należy:

  1. Z menu Edycja &brandShortName; wybrać Preferencje.
  2. W kategorii Kurier Poczty wybrać Potwierdzenia. (Jeśli podkategorie nie są widoczne, to aby rozwinąć listę, należy podwójnie kliknąć na Kurier Poczty.)
  3. Zaznaczyć Podczas wysyłania wiadomości zawsze żądaj potwierdzenia dostarczenia wiadomości.
  4. Nacisnąć przycisk OK.

Więcej informacji o wysyłaniu potwierdzeń otwarcia (dostarczenia) wiadomości w rozdziale Preferencje Kuriera Poczty — Potwierdzenia.

Zapisywanie i edycja szkiców wiadomości

Aby zachować tworzoną wiadomość jako szkic, w celu późniejszej edycji, należy:

  • W oknie tworzenia wiadomości nacisnąć przycisk Zapisz lub z menu Plik wybrać Zapisz jako oraz wskazać Szkic. Domyślnie wiadomość jest zapisywana w folderze Szkice bieżącego konta.

    Uwaga: Okno tworzenia wiadomości pozostanie otwarte po zapisaniu wiadomości jako szkic.

Aby edytować lub wysłać szkic wiadomości, należy:

  1. Otworzyć folder Szkice konta, gdzie została stworzony szkic wiadomości.
  2. Wybrać wiadomość, która ma być edytowana.
  3. Nacisnąć przycisk Edytuj szkic w prawym górny rogu wiadomości.
  4. Edytować wiadomość.
  5. Nacisnąć przycisk Wyślij, aby wysłać wiadomość lub nacisnąć przycisk Zapisz, aby zachować wiadomość do późniejszej edycji.

    Uwaga: Po wysłaniu wiadomość jest usuwana z folderu Szkice.

Podpowiedź: Aby rozpocząć edycję wiadomości można dwukrotnie kliknąć na wybranej wiadomości. Jest to szczególnie przydatne, kiedy panel wiadomości jest zamknięty.

Aby usunąć niepotrzebne szkice wiadomości, należy:

  1. Otworzyć folder Szkice konta, gdzie znajdują się szkice wiadomości.
  2. Wybrać wiadomości do usunięcia.
  3. Nacisnąć przycisk Usuń.

Tworzenie i korzystanie z szablonów

Szablony są przydatne przy tworzeniu wiadomości, które są wysyłane regularnie, jak na przykład tygodniowe raporty. Wiadomość może zostać zapisana jako szablon z każdego okna, w którym jest wyświetlona, włączając okno tworzenia wiadomości.

Aby zachować wiadomość jako szablon, należy:

  1. W oknie Kuriera Poczty nacisnąć przycisk Nowa, wpisać treść szablonu, gdy zachodzi potrzeba ustawić kolory, czcionki itp.

    Można także otworzyć istniejącą wiadomość.

Używanie HTML w wiadomościach

Wiadomości w formacie HTML mogą zawierać sformatowany tekst, odnośniki, obrazki, tabele oraz inne elementy języka HTML. Jednakże należy pamiętać, że nie wszystkie programy pocztowe obsługują format HTML. Kurier Poczty &brandShortName; umożliwia wybór edytora HTML lub tekstowego, jako domyślnego przy tworzeniu wiadomości. Wybór jest dostępny dla każdego z kont osobno (w ustawieniach Konfiguracja konta). Ponadto możliwe jest ustalenie domyślnego formatu wiadomości dla poszczególnych adresatów wiadomości (Książka adresowa), a także jakich formatów należy użyć w przypadku, gdy nie jest pewne, czy program pocztowy danego adresata może poprawnie wyświetlać wiadomości w HTML.

Aby określić domyślny edytor używany przy tworzeniu wiadomości, należy:

  1. Z menu Edycja wybrać Konfiguracja kont.
  2. Wybrać konto pocztowe lub konto grup dyskusyjnych, którego domyślny edytor ma zostać zmieniony.
  3. W podkategorii Komponowanie zaznaczyć opcję Używaj edytora HTML do tworzenia wiadomości. W oknie tworzenia wiadomości zostanie wyświetlony pasek narzędzi formatowania. Aby używać zwykłego edytora dla wybranego konta, należy odznaczyć wspomnianą opcję.

Edycja lub wstawianie elementów HTML

Kliknąć przycisk Zaawansowana edycja, aby otworzyć okno Zaawansowanego edytora właściwości, w którym można dodać do elementu atrybuty HTML lub skrypty JavaScript.

Kurier Poczty umożliwia wybranie domyślnego sposobu wysyłania wiadomości jeżeli nie jest pewne, czy program pocztowy odbiorcy może poprawnie wyświetlać wiadomości w formacie HTML.

Aby zmienić ustawienia wysyłania wiadomości, należy:

  1. Z menu Edycja &brandShortName; wybrać Preferencje.

Jeżeli w trakcie tworzenia wiadomości okaże się, że jeden lub więcej odbiorców nie może odczytywać wiadomości w formacie HTML, możliwa jest łatwa zmiana formatu wiadomości po naciśnięciu przycisku Wyślij:

    • Tylko tekst: Część formatowania, jak np. pogrubienie tekstu zostanie utracone, lecz wiadomość zostanie wyświetlona poprawnie przez wszystkie programy pocztowe
    • Tylko HTML: Niektóre programy pocztowe mogą nie wyświetlić wiadomości poprawnie. Z opcji należy korzystać tylko gdy jest pewność, że odbiorca jest ma możliwość odczytania wiadomości w formacie HTML.
    • Tekst oraz HTML: Wysyłana wiadomość ma większy rozmiar, jednakże jest to najlepszy wybór w przypadku, gdy użytkownik nie jest pewien czy odbiorca ma możliwość odczytania wiadomości w formacie HTML.
  3. Po zakończeniu edytowania wiadomości należy nacisnąć przycisk Wyślij, a Kurier Poczty wyśle wiadomość w wybranym formacie.

Przeglądanie i otwieranie załączników

Jeżeli otrzymane wiadomości zawierają załączniki w formacie obsługiwanym przez Kuriera Poczty, zostaną one wyświetlone w treści wiadomości (np. pliki graficzne lub HTML). W przypadku otwierania załączników, których format nie jest bezpośrednio obsługiwany, Kurier Poczty uruchomi aplikację, która została uprzednio powiązana z danym typem plików lub umożliwi zapisanie załączników na dysku komputera.

Aby otworzyć załącznik, którego format nie jest obsługiwany bezpośrednio, należy upewnić się, że została poprawnie skonfigurowana aplikacja, która potrafi otworzyć pliki takiego samego formatu, jak załącznik. Na przykład, aby otworzyć pliki zapisane w formacie .DOC, należy upewnić się, że w komputerze istnieje aplikacja, która potrafi otworzyć pliki .DOC.

Aby otworzyć załącznik, należy:

  1. Kliknąć dwukrotnie na danym załączniku (jeżeli jest więcej niż jeden).
  2. W wyświetlonym oknie dialogowym wybrać żądaną akcję dla pliku:
    • W przypadku, gdy &brandShortName; znajdzie w komputerze użytkownika aplikację obsługującą dany typ pliku, można otworzyć załącznik z jej pomocą. Aby otworzyć załącznik za pomocą innej aplikacji, należy nacisnąć przycisk "Wybierz".
    • W przypadku, gdy w komputerze użytkownika nie została znaleziona aplikacja obsługująca dany typ pliku, załącznik można zapisać na dysku.
    • Aby dodać nowy typ plików do listy aplikacji pomocniczych, należy nacisnąć przycisk Zaawansowane. &brandShortName; wykorzystuje aplikacje pomocnicze do otwierania nieobsługiwanych typów plików. Więcej informacji w rozdziale Wtyczki i pobieranie plików.
  3. Nacisnąć przycisk OK.

Uwaga: W przypadku przeglądania załączników dla konta typu IMAPI, wszystkie załączniki pozostają na serwerze.

Książki adresowe — wprowadzenie

W książkach adresowych przechowywane są informacje takie jak adresy e-mail, dane osobowe, numery telefonów itp., o osobach, do których często wysyłane są wiadomości. Kurier Poczty posiada wbudowane dwie książki adresowe: Osobista książka adresowa oraz Zebrane adresy. Ponadto można utworzyć dowolną ilość własnych książek adresowych. Zawartość wszystkich książek przechowywana jest lokalnie na dysku komputera.

Książka adresowa może również zawierać adresy z serwera usług katalogowych LDAP. Serwer katalogowy przechowuje adresy osób których nie ma w lokalnych książkach adresowych. Protokół LDAP (Lightweight Directory Access Protocol) służy do dostępu do usług katalogowych (takich jak firmowe książki adresowe) proprzez sieć lokalną lub Internet.

Osobista książka adresowa

W Osobistej książce adresowej można zapisywać dane adresatów lub importować dane z książek adresowych w innych programach. Możliwe jest tworzenie list wysyłkowych oraz edycja danych poszczególnych odbiorców.

Zebrane adresy

Do książki adresowej Zebrane adresy automatycznie dodawane są adresy zawarte w wysyłanych wiadomościach. Adresy z wiadomości wysyłanych dodawane są zaraz po naciśnięciu przycisku Wyślij.

Usługi katalogowe LDAP (jeżeli dostępne)

Usługa katalogowa LDAP (nazywana często usługą wyszukiwania adresów) umożliwia dostęp do adresów odbiorców, których dane nie są przechowywane w lokalnych książkach adresowych. Usługa katalogowa LDAP udostępnia duże, centralnie zarządzane bazy danych adresowych, które są szczególnie przydatne przy wykorzystywaniu funkcji autouzupełniania adresów.

Automatyczne zbieranie adresów jest domyślnie włączone. Aby zmienić ustawienia dla zbierania adresów, należy:

  1. Z menu Edycja &brandShortName; wybrać Preferencje.
  2. W kategorii Kurier Poczty wybrać Adresowanie. (Jeśli podkategorie nie są widoczne, to aby rozwinąć listę, należy podwójnie kliknąć na Kurier Poczty.)
  3. W polu Zbieranie adresów e-mail należy odznaczyć opcję Dodawaj adresy do książki adresowej i wybrać książkę adresową, w której zebrane adresy będą zachowywane:
    • Osobista książka adresowa.
    • Zebrane adresy.
  4. Nacisnąć przycisk OK.

Tworzenie nowej wizytówki w książce adresowej

Dodawanie i usuwanie katalogów LDAP

Dodanie katalogu LDAP do książki adresowej daje możliwość wyszukiwania adresów e-mail i innych danych kontaktowych w nim zawartych. Umożliwia także autouzupełnianie adresów e-mail podczas adresowania wiadomości.

Zazwyczaj dodawanie i usuwanie serwerów usług katalogowych powinno odbywać się zgodnie z instrukcjami administratora systemu, od którego należy uzyskać niezbędne dane serwera.

Aby dodać nowy serwer usług katalogowych LDAP, należy w oknie Książka adresowa:

  1. Z menu Plik wybrać Nowy, a następnie katalog LDAP. Wyświetli się okno Właściwości serwera usług katalogowych.
  2. W karcie Ogólne wprowadzić następujące informacje:
    • Nazwa: Nazwę serwisu usługi katalogowej (np. InfoSpace Directory).
    • Nazwa Hosta: Nazwa lub adres serwera udostępniającego usługę, np ldap.infospace.com.
    • Bazowy DN: Kod kraju lub organizacji, ograniczający zakres wyszukiwania. Na przykład: c=PL ograniczy wyszukiwanie do danych z Polski. Za pomocą bazowego DN możliwe jest ograniczenie wyszukiwania do obrębu danej organizacji (na przykład: o=Netscape Communications Corporation, c=US spowoduje, że wyszukiwane będą wyłącznie adresy firmy Netscape w Stanach Zjednoczonych). Aby pobrać z serwera informacje o Bazowym DN oraz wyświetlić listę dostępnych kodów, należy nacisnąć przycisk Znajdź.
    • Numer Portu: Numer portu, na którym udostępniony jest serwer usług katalogowych LDAP. Domyślnie jest to port numer 389.
    • Serwer Bind DN: Identyfikator używany w celu uwierzytelnienia (zalogowania) na serwerze LDAP. Jeżeli pole pozostanie puste program spróbuje uwierzytelnić się anonimowo.
    • Użyj szyfrowanego połączenia (SSL): Należy wybrać, jeżeli serwer LDAP obsługuje bezpieczne (szyfrowane) połączenia. W razie wątpliwości należy skontaktować się z administratorem systemu.
  3. Wybrać kartę Zaawansowane, aby ustawić dodatkowe opcje
  4. Należy wprowadzić następujące informacje:
    • Nie zwracaj więcej niż _ wyników: To ustawienie pozwala na ograniczenie liczby adresów zwracanych podczas autouzupełniania adresu przez serwer usług katalogowych. Należy wprowadzić maksymalną ilość adresów e-mail do wyświetlenia podczas autouzupełniania.
    • Obszar wyszukiwania: definiuje limity wyszukiwania. Należy wybrać jedną z poniższych opcji:
      • Jeden poziom: Zwraca wyniki dopasowania pochodzące z wyszukiwania bazowego DN oraz jednego poziomu poniżej bazowego DN.
      • Wszystkie poziomy: Zwraca wyniki dopasowania pochodzące z wyszukiwania bazowego DN oraz wszystkich poziomów poniżej bazowego DN. Ten rodzaj wyszukiwania jest najmniej restrykcyjny.
    • Filtr wyszukiwania: W to pole można wprowadzić dodatkowy filtr, który zostanie zastosowany do wyników, po uwzględnieniu maksymalnej liczby dopasowań.
  5. Nacisnąć przycisk OK, aby zamknąć okno Właściwości serwera usług katalogowych.
  6. Nacisnąć przycisk OK, aby zamknąć okno Serwera usług katalogowych.
  7. Nacisnąć przycisk OK, aby zamknąć okno ustawień.

Dodany katalog pojawi się na liście książek adresowych w oknie książki adresowej.

Aby usunąć katalog, należy:

  1. Z menu Edycja &brandShortName; wybrać Preferencje.
  2. W kategorii Kurier Poczty wybrać Adresowanie. (Jeśli podkategorie nie są widoczne, to aby rozwinąć listę, należy podwójnie kliknąć na Kurier Poczty.)
  3. W polu autouzupełnianie adresu, należy wybrać Edytuj katalogi.
  4. W oknie dialogowym Serwery usług katalogowych LDAP wybrać serwer usług katalogowych, który ma zostać usunięty, a następnie nacisnąć przycisk Usuń.
  5. Nacisnąć przycisk OK, aby zamknąć okno dialogowe właściwości serwera usług katalogowych.

Tworzenie filtrów wiadomości

Filtry wiadomości pozwalają na łatwe organizowanie zarządzania wiadomościami.   3. Wybrać Nowy. W oknie Reguły filtrów należy określić reguły, według których filtr będzie sortował wiadomości oraz wykonywał na nich operacje.
  4. Podać nazwę dla filtru.
  5. Wybrać odpowiednie opcje: spełniają wszystkie poniższe kryteria lub spełniają dowolne poniższe kryteria.
  6. Z rozwijanej listy należy wybrać krytria przeszukiwania wiadomości (np. Temat, Nadawca, zawiera, nie zawiera), a następnie wpisać warunek przeszukiwania.

Dobrze jest się upewnić, czy wprowadzony nagłówek jest poprawny, ponieważ program wyszuka tylko wiadomości dokładnie spełniające wprowadzony warunek.

    Wskazówka: Aby automatycznie nadawać etykiety przychodzącym wiadomościom, należy z rozwijanej listy wybrać Nadaj etykietę.

  9. Wybrać docelowy folder dla wiadomości lub stworzyć nowy.
  10. Nacisnąć przycisk OK, aby zatwierdzić ustawienia.
  11. Nacisnąć przycisk OK w oknie fitrów wiadomości. Filtr zacznie działać w momencie potwierdzenia przyciskiem OK.

Aby zarządzać filtrami, w oknie Kuriera Poczty należy:

  1. Z menu Narzędzia wybrać Filtrowanie wiadomości.
  2. W przypadku, gdy skonfigurowano więcej kont pocztowych, wybrać właściwe dla zastosowania filtrów.
  3. Wybrać z poniższych:

    • Aby zmienić kolejność w jakiej fitry są aplikowane: Na liście filtrów należy wybrać filtr i nacisnąć przycisk Przenieś w górę lub Przenieś w dół.

Uwaga: Po usunięciu folderu, do którego filtr przenosił wiadomości, to filtr ten przestanie działać. Przychodzące wiadomości spełniające określone w nim kryteria pozostaną w folderze Odebrane. Zmiana nazwy folderu lub zmiana jego położenia spowoduje automatyczną aktualizację parametrów filtru, aby wciąż używał tego folderu.

Wskazówka: Aby przenieść wiadomości do innych folderów za pomocą filtrów należy użyć opcji Zastosuj filtry w bieżącym folderze dostępnej w menu Narzędzia.

Kurier Poczty &brandShortName; umożliwia szybkie wyszukanie tekstu w pojedynczej wiadomości, wyszukiwanie wiadomości po temacie lub nadawcy albo użyć kombinacji kryteriów, aby dokładnie przeszukać wszystkie wiadomości w określonym folderze, grupie dyskusyjnej lub koncie.

Aby wyszukać tekst w pojedynczej wiadomości, w oknie Kuriera Poczty należy:

  1. Wybrać wiadomość, z menu Edycja wybrać Znajdź w tej wiadomości.
  2. Wprowadzić w oknie dialogowym tekst do wyszukania.
  3. Nacisnąć przycisk Znajdź, aby zlokalizować pierwsze wystąpienie wyszukiwanego tekstu.
  4. Kolejne naciskanie przycisku Znajdź spowoduje lokalizowanie kolejnych wystąpień wyszukiwanego tekstu. Naciśnięcie przycisku Anuluj zakończy wyszukiwanie.
  5. Wybór z menu Edycja opcji Znajdź następne umożliwia kontynuowanie wyszukiwania tekstu w pozostałej części wiadomości.

Aby szybko wyszukać w wybranym folderze wiadomości od określonego nadawcy lub o określonym temacie, w oknie Kuriera Poczty należy:

  1. Z prawej strony napisu Temat lub nadawca zawiera: wprowadzić tekst lub nazwę nadawcy do wyszukania. Można wprowadzić tylko część wyrażenia lub dokładą treść do wyszukania.

    Po zakończeniu wprowadzania tekstu &brandShortName; wyświetli tylko te wiadomości w folderze, których temat lub nadawca zawiera wpisany tekst.

  2. Nacisnąć przycisk Wyczyść, aby usunąć tekst do wyszukania i wyświetlić wszystkie wiadomości w aktualnym folderze.

Wyszukiwanie konkretnych wiadomości

Można wyszukać określone wiadomości w folderach z wiadomościami pocztowymi lub w wiadomościach z grup dyskusyjnych. Aby rozpocząć, należy w oknie Kuriera Poczty:

  1. Z menu Narzędzia wybrać Wyszukaj wiadomości. Wyświetlone zostanie okno dialogowe Wyszukaj wiadomości.
  2. Obok Wyszukaj wiadomości, które należy wybrać z listy konto, grupę dyskusyjną lub folder, w którym ma być prowadzone wyszukiwanie.
  3. Zaznaczyć Kontynuuj wyszukiwanie w podfolderach, aby wyszukiwać również we wszystkich podfolderach.
  4. Zaznaczyć Wyszukaj w lokalnym systemie, aby wyszukiwać jedynie w zapisanych lokalnie wiadomościach z grup dyskusyjnych lub z kont IMAP.

    Uwaga: Opcja będzie niedostępna, jeżeli nie będzie możliwe wyszukiwanie w wiadomościach przechowywanych na serwerze.

  5. Wybrać, która z opcji dopasowywania warunków będzie używana w czasie wyszukiwania, spełnianie wszystkich lub dowolnego z kryteriów.
  6. Użyć rozwijalnych list, aby wybrać kryteria wyszukiwania (przykładowo Temat i zawiera) i następnie wpisać tekst lub frazę do wyszukania.

    Wskazówka: Aby wyszukiwać wiadomości, których nagłówek nie jest zdefiniowany na rozwijalnej liście (przykładowo wyszukiwanie wiadomości, których zawierają nagłówek Resent-From (przesłane z), należy wybrać Dostosuj i wpisać nagłówek do wyszukiwania. Kurier Poczty doda go do rozwijalnej listy i będzie możliwe wybranie go do określenia kryterium wyszukiwania. Należy się upewnić, że nagłówek został wprowadzony poprawnie, ponieważ wyszukiwanie będzie prowadzone dokładnie według wprowadzonego tekstu.

  7. Nacisnąć przycisk + aby dodać kolejne kryterium lub przycisk - aby usunąć kryterium.
  8. Nacisnąć przycisk Znajdź, aby rozpocząć wyszukiwanie lub nacisnąć przycisk Wyczyść, aby usunąć kryteria wyszukiwania. Wyniki wyszukiwania znajdą się w dolnej części okna Wyszukaj wiadomości.
    • Aby wyświetlić wyszukaną wiadomość, należy ją zaznaczyć i nacisnąć przycisk Otwórz lub podwójnie kliknąć na niej.
    • Aby posortować wiadomości w innym porządku, należy kliknąć nagłówek kolumny, po której ma być przeprowadzone wyszukiwanie.
    • Aby przenieść lub skopiować wyszukane wiadomości z wyników wyszukiwania do innego folderu, należy zaznaczyć wiadomości i wybrać folder docelowy z rozwijalnej listy dostępnej po naciśnięciu przycisku Przenieś. Jeżeli folder docelowy jest w obrębie tego samego konta, to wiadomość jest przenoszona do niego. Jeżeli folder należy do innego konta, to wiadomość jest do niego kopiowana.
    • Aby usunąć wyszukaną wiadomość, należy ją zaznaczyć i nacisnąć przycisk Usuń.
    • Aby wyświetlić folder, w którym wiadomość jest przechowywana, należy zaznaczyć wiadomość i nacisnąć przycisk Otwórz folder wiadomości.

Filtr niechcianej poczty

Filtr niechcianej poczty &brandShortName; może oceniać przychodzące wiadomości i identyfikować wiadomości jako najprawdopodobnie niepożądane (niezamówione reklamy). Używana do tego jest metoda Bayesa. Najpierw należy nauczyć &brandShortName; jak wyglądają wiadomości niechciane wskazując, które wiadomości są niepożądane, a które pożądane. Następnie włącza się automatyczną klasyfikację wiadomości. Jeżeli &brandShortName; zrobi błąd w rozpoznaniu wiadomości, to można go poprawić.

Aby używać filtru niechcianej poczty należy:

  1. Najpierw nauczyć &brandShortName; rozpoznawać wiadomości niechciane i pożądane. Odpowienie oznaczenie wiadomości można zmieniać trzema sposobami:
    • Z menu Wiadomość wybrać Zaznacz i odpowiednio Jako niechcianą lub Jako pożądaną.
    • Nacisnąć przycisk na pasku narzędziowym Niechcianej poczty.

    • Kliknąć na znaczku w kolumnie statusu niechcianej poczty na liście wiadomości. (Jeżeli kolumna nie jest widoczna, należy nacisnąć przycisk na prawym końcu nagłówka listy wiadomości () i wybrać z menu pozycję Status niechcianej poczty.)

    Po zmianie statusu wiadomości w kolumnie statusu niechcianej poczty pojawi się lub zniknie ikona kosza odpowiednio wskazując status wybranej wiadomości.

  2. Z menu Edycja wybrać Konfiguracja kont. Wyświetlone zostanie okno dialogowe Konfiguracja kont.
  3. Wybrać kategorię Ustawienia niechcianej poczty dla konta pocztowego.
  4. Po włączeniu funkcji filtrowania &brandShortName; będzie automatycznie klasyfikowała przychodzące wiadomości. (Więcej w rozdziale Ustawienia filtru niechcianej poczty. Szczegóły odnośnie innych ustawień można znaleźć w opisie panelu preferencji Ustawienia niechcianej poczty.)
  5. Jeżeli uczenie było prowadzone na wiadomościach z wirusami, należy rozważyć wyłączenie białych list, ponieważ wiele wirusów wysyła zainfekowane wiadomości do osób z książki adresowej zainfekowanego komputera.
  6. Należy poprawiać status niechcianej poczty dla nieprawidłowo oznaczonych wiadomości, zarówno niepożądanych jak i pożądanych.
  7. Aby przeprowadzić analizę odebranych wcześniej wiadomości, należy wybrać te wiadomości i wybrać z menu Narzędzia opcję Rozpoznaj w folderze niechciane wiadomości.

Uwaga: &brandShortName; uruchomi filtrowanie niechcianej poczty tylko jeżeli baza wiadomości służących do uczenia zawiera również pożądane wiadomości. Jeżeli filtrowanie nie działa, należy wybrać kilka wiadomości i oznaczyć je jako pożądane.

Wykrywanie oszustw

Podszywanie się (phishing) jest stosunkowo częstym schematem wyłudzania, w którym osoba lub grupa osób tworzy fałszywe strony WWW przeznaczone do oszukania adresatów wiadomości tak, aby poznać ich dane osobowe takie jak numery kart kredytowych, nazwy użytkowników kont, hasła, numery PESEL. Używając nazw i marek banków, sieci handlowych oraz firm wydających karty kredytowe wyłudzający często uzyskują pożądane informacje od nieświadomych odbiorców takich wiadomości.

Można otrzymać odnośnik do strony podszywającej się w wiadomości e-mail, która wygląda na wysłaną z oficjalnego adresu danej firmy. Na taką stronę można również trafić aktywując odnośnik otrzymany w treści wiadomości przesłanej za pomocą komunikatora lub na stronach WWW.

Wskazówka: Jako, że sfałszowany URL może wyglądać bardzo podobnie do oryginalnego, bezpieczniej jest używać własnoręcznie utworzonych zakładek lub wprowadzać adres URL ręcznie w pasku adresu zamiast aktywować odnośnik z wiadomości e-mail. Zawsze należy rozważyć możliwość sfałszowania adresu URL, gdy strona prosi o zalogowanie się lub podanie prywatnych danych.

Wykrywanie oszustw w Kurierze Poczty &brandShortName; jest domyślnie włączone. Po napotkaniu wiadomości, która wygląda na próbującą oszukać użytkownika (scam), w oknie wiadomości wyświetlony zostanie pasek z ostrzeżeniem.

Jeżeli użytkownik uważa, że wiadomość jest prawdziwa, może nacisnąć klawisz Prawdziwa i pasek z ostrzeżeniem zostanie wyłączony.

Gdy użytkownik kliknie w wiadomości na odnośniku wyglądającym na sfałszowany &brandShortName; wyświetli okno dialogowe z prośbą o potwierdzenie zanim strona WWW zostanie wyświetlona.

Okno to zostanie wyświetlone, gdy wystąpi jedna z okoliczności: nazwę hosta w odnośniku stanowi adres IP lub tekstowy URL jest nazwą hosta niezgodną z nazwą hosta podaną w odnośniku.

Uwaga: Wykrywanie oszustw ma pierszeństwo przed filtrem niechcianej poczty.

Więcej informacji technicznych w dokumencie Know your Enemy: Phishing, z angielskiego: Poznaj swojego wroga - Podszywanie się.

Subskrybowanie grup dyskusyjnych

Jeżeli użytkownik ma ustawione konto na serwerze grup dyskusyjnych, to może dołączyć do (rozpocząć subskrybcję) grupy dyskusyjnej (również nazywanej grupą niusową).

Aby rozpocząć subskrypcję grupy dyskusyjnej, należy w oknie Kuriera Poczty:

  1. Z menu Plik wybrać Subskrybuj. Wyświetlone zostanie okno dialogowe Subskrybuj.
  2. Jeżeli jest to konieczne, należy wybrać inne konto z listy rozwijanej.
  3. Wybrać grupę dyskusyjną. Aby wybrać więcej niż jedną grupę, należy trzymać wciśnięty klawisz CmdCtrl w czasie wybierania kolejnej grupy.
  4. Nacisnąć przycisk Subskrybuj lub kliknąć w kolumnie Subskrybuj obok nazwy grupy. Obok każdej wybranej do subskrybcji grupy wyświetlony zostanie znaczek zaznaczenia. Naciśnięcie klawisza Anuluj subskrypcję powoduje rezygnację z subskrybowania wybranej grupy.
  5. Nacisnąć przycisk OK. Lista wybranych do subskrybowania grup pojawi się w oknie Kuriera Poczty.

Jeżeli użytkownik używa serwera IMAP, może również subskrybować foldery z wiadomościami na serwerze IMAP. (Folder Odebrane jest typem folderu wiadomości.) Należy postępować zgodnie z instrukcjami subskrybowania powyżej, ale wybrać z listy rozwijalnej konto na serwerze IMAP. IMAP - Internet Message Access Protocol

Zalety: Wiadomości i zmiany w nich dokonywane są przechowywane na serwerze, dzięki temu nie zajmują miejsca na dysku lokalnego komputera. Równocześnie użytkownik ma zawsze zapewniony dostęp do aktualnej zawartości skrzynki pocztowej oraz może pobierać wiadomości w wielu miejscach. Połączenia przez modem telefoniczny są szybsze, ponieważ początkowo pobierane są tylko nagłówki.

Wady: Nie wszyscy dostawcy usług zapewniają obsługę IMAP.

[Na początek rozdziału]

POP - Post Office Protocol

Zalety: Wiadomości są pobierane od razu na dysk lokalnego komputera w całości, ale równocześnie można określić czy kopie wiadomości mają być przechowywane na serwerze i usuwane z niego po ich usunięciu na komputerze użytkownika. Większość dostawców usług internetowych zapewnia obsługę POP.

Wady: Jeżeli użytkownik używa więcej niż jednego komputera, to wiadomości mogą być przechowywane na jednym lub drugim komputerze, ale nie na obu jednocześnie. Przy wolnym połączeniu POP nie działa równie wydajnie jak IMAP. Nie ma dostępu do wszystkich folderów pocztowych z wielu miejsc.

Należy zauważyć, że większość współczesnych serwerów POP umożliwia pobieranie tylko nagłówków wiadomości zamiast całych wiadomości, podobnie jak IMAP. Praca z wykorzystaniem tego trybu zapewnia POP praktycznie taką samą wydajność jak IMAP.

The main purpose of the Security panel in Mail & Newsgroup Account Settings is to select two certificates:

Depending on the policies of the certificate authority (CA) that issues your certificate(s), you can use one certificate for both purposes or two different certificates. Even if you use just one, you must specify it twice, once for digital signing and once for encryption.

Formatowanie

Send Format preferences allow you to specify how you want to format your outgoing messages. If you are not already viewing the Send Format settings, follow these steps:

  1. Z menu Edycja &brandShortName; wybrać Preferencje.
  2. W kategorii Kurier Poczty wybrać Formatowanie. (Jeśli podkategorie nie są widoczne, to aby rozwinąć listę, należy podwójnie kliknąć na Kurier Poczty.)
  • Ask me what to do: This option requires Mail & Newsgroups to prompt you to choose a format before you send the message.
  • Convert the message to plain text: This option may cause your message to lose formatting such as bold text.
  • Send the message in HTML anyway: If you select this option, keep in mind that some mail programs may have trouble displaying the message.
  • Send the message in both plain text and HTML: This option uses more disk space.

    You can always override these preferences for an individual message by using the Options menu in the Mail Compose window.

  • HTML and Plain Text Domains: Use the Add button to add the domain names that you typically send mail to, if you know which domains can display HTML-formatted mail messages, and which domains can only display plain text.

For example, if you typically send mail to multiple recipients that have the same domain name (for example, your colleagues all have email addresses that end in netscape.net), and you know that this domain name is capable of displaying HTML messages, then you can add the netscape.net domain to the list of HTML Domains so that Mail & Newsgroups will automatically send messages in HTML format to these recipients.

Similarly, if you typically send mail to recipients at a domain that you know can only receive Plain Text messages, you can add that domain name to the list of Plain Text domains, so that Mail & Newsgroups automatically sends messages to that domain in plain-text format.

Uwaga: If you regularly compose HTML (formatted) mail messages, keep in mind that sometimes not all recipients use mail programs that can display HTML formatting properly. Send Format preferences allow you to specify how you want to format messages that go to recipients who cannot display HTML-formatted mail. You can convert messages to plain text, format them only as HTML, or format them as both HTML and plain text. These preferences apply to all your mail accounts, but only to mail messages and not to newsgroup messages.

Whenever you add a person or address card to your address book, you can specify whether that addressee can receive HTML-formatted messages. However, when this information is unknown, you can set Send Format preferences for how Mail & Newsgroups formats these messages.
