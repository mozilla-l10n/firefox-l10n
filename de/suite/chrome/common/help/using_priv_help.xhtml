Anzeigen von Datenschutz-Einstellungen

Wenn Sie persönliche Informationen wie Name, Telefon-Nummer oder Email-Adresse einer Website bekannt geben, oder die Website diese auf andere Art sammelt, kann diese Site diese Inforamtionen für Ihre eigenen Zwcke gebrauchen (zum Beispiel Online-Bestellungen) aber auch diese Daten mit anderen teilen.

Bevor Sie persönliche Informationen in ein Formular eintragen, müssen Sie entscheiden ob Sie der Website vertrauen oder nicht—ähnlich wie Sie entscheiden müssen, ob Sie einer Katalog-Firma vertrauen, bevor Sie Ihre Kreditkarten-Nummer auf der Bestellung angeben.

Eine Möglichkeit die Vertrauenswürdigkeit einer Website zu untersucht, ist deren Datenschutz-Richtlinien anzusehn. Websites veröffentlichen ihre Datenschutz-Richtlinien online, manche benutzen dazu den Plattform für Datenschutzeinstellungen (P3P) - Standard.

Die Datenschutz-Richtlinien einer Website beschreiben welche Art von Informationen diese Seite sammelt, an wen diese diese Informationen weitergegeben werden und wie diese Daten genutzt werden. Websites veröffentlichen Ihre Datenschutz-Richtlinien sowohl in einer für Menschen lesbaren Form als auch in Form eines Files, welches von Ihrem Webbrowser gelesen und entsprechend Ihren Einstellungen zum Datenschutz interpretiert wird.

Um die Datenschutz-Richtlinien einer Webseite anzuzeigen, besuchen Sie diese Seite und folgen Sie diesen Schritten:

  • Öffnen sie das Menü Ansicht und wählen Sie Seiteninformationen. Das Seiteninformationen-Fenster erscheint.
  • Klicken Sie auf die Registerkarte Datenschutz.
  • Klicken sie auf einen der Knöpfe um die entsprechenden Informationen zu sehn:
    • Richtlinien: Klicken Sie diesen Button um die Datenschutz-Richtlinien der Seite im Web anzuzeigen.
    • Zusammenfassung: Klicken sie diesen Button, um eine Zusammenfassung des XML-Files welches die Datenschutz-einstellungen der Website beinhaltet anzuzeigen.
    • Optionen: Klicken Sie diesen Button um zu einer Webseite zu gelangen, welche es Ihnen ermöglicht, Ihre Zustimmung oder Ablehnung zur Informationssammlung der Site zu geben.

Viele Webseiten sind aus verschiedenen Stücken an Text, Bildern und anderen Elementen zusammengestellt, manche von diesen Elementen können von anderen Firmen kontrolliert werden.

Um die Informationen zu jedem Einzelnem Element auf der Webseite zu sehn, folgen Sie diesen Anweisungen:

  1. Doppelklicken Sie auf die Kategorie des gewählten Elements.
  2. Wählen Sie das gewünschte Element dessen Datenschutz-Richtlinien Sie ansehn möchten.

Verschlüsseln sensibler gespeicherter Informationen

Wenn Sie den Passwort-Manager oder den Formular-Manager zum Speichern von Passwörtern und persönlichen Daten nutzen, werden diese Informationen auf Ihrem Computer in einer Datei gespeichert, die schwer, aber nicht unmöglich, von einem Hacker gelesen werden kann.

Wenn ihr Computer zum Beispiel an einem Platz steht, wo nicht berechtigte Personen Zugriff haben, ist es möglich, dass diese Personen die Datei mit den sensibler Informationen lesen.

Um eine grössere Sicherheit zu gewährleisten, können Sie die Datei zusätzlich noch verschlüsseln. Diese Verschlüsselung macht es nicht berechtigten Personen um einiges schwerer, Ihre sensibler Daten zu lesen.

Ihre Entscheidung die gespeicherten sensiblen Daten zu verschlüsseln, stellt zwar eine Verbesserung der Sicherheit dar, ist jedoch auch mit einer Komfort-Einbuse verbunden.

Wenn Sie die Verschlüsselung nutzen, werden Sie in bestimmten Zeitabständen dazu aufgefordert, ein sogenanntes Master-Passwort einzugeben, was unbequem sein kann. Wenn Sie die Verschlüsselung jedoch nicht nutzen, ist es für Fremde leichter, Ihre Passwörter zu stehlen.

Ein Master-Passwort erstellen

Wenn Sie die Verschlüsselung aktivieren, aber noch kein Master-Passwort besitzen, werden Sie beim ersten mal wo Sie auf gespeicherte sensible Daten zugreifen oder solche speichern dazu aufgefordert, eines anzulegen.

Vom Master-Passwort abmelden

Normalerweise werden Sie einmal pro Browser-Sitzung dazu aufgefordert Ihr Master-Passwort anzugeben, wenn Sie auf gespeicherte sensible Daten zugreifen möchten.

Es ist auch möglich, einzustellen, dass Sie jedes Mal nach dem Master-Passwort gefragt werden, oder nach Ablauf einer bestimmten Zeitspanne. Für Details siehe Master-Passwort-Gültigkeitsdauer.

Sie können sich aus Ihrem Master-Passwort abmelden, so dass Sie es beim nächsten Mal, wo Sie auf sensible gespeicherte Daten zugreifen erneute eingeben müssen. Dies ist nützlich, wenn Sie Ihren Computer für einige Zeit unbeaufsichtigt lassen.

Um sich von Ihrem Master-Passwort abzumelden folgen Sie diesen Schritten:

  1. Öffnen Sie das Menü Extras und wählen Sie Passwort-Manager.
  2. Wählen Sie Ausloggen aus dem Untermenü.

Was müssen Sie tun, wenn Sie Ihr Master-Passwort vergessen haben

Wenn Sie Ihr Master-Passwort vergessen haben und auf sensible gespeicherte Daten zugreifen wollen (siehe Verschlüsseln sensibler gespeicherter Informationen), wird Ihnen der Zugriff auf die gespeicherten Passwörter und Formular-Daten verweigert (sofern Sie die Verschlüsselung aktiviert haben). Ihr Master-Passwort ist ihr wichtigstes Passwort. Stellen Sie sicher dass Sie es nicht vergessen oder notieren Sie es sich an einem sicherem Ort.

Als letzten Ausweg bleibt Ihnen die Möglichkeit das Master-Passwort zurückzusetzen. Verwenden Sie diese Möglichkeit erst, wenn Sie sich sicher sind, dass Sie sich nicht an Ihr Master-Passwort erinnern. Beachten Sie, dass das Zurücksetzen Ihres Master-Passworts zur Folge hat, dass all Ihre Passwörter und Formular-Daten welche im Passwort-Manager und Formular-Manager gespeichert wurden, unwiderruflich gelöscht werden. Sie verlieren ebenfalls all Ihre persönlichen Zertifikate, welche mit dem Software-Kryptographie-Modul verbunden sind.

Bevor Sie diesen drastischen Schritt unternehmen, lesen Sie Master-Passwort rücksetzen.

Wenn Sie sicher sind, dass Sie sich nicht and Ihr Master-Passwort erinnern können, folgen Sie diesen Anweisungen:

  1. Öffnen Sie das Menü Bearbeiten&brandShortName;, klicken Sie auf Einstellungen.
  2. Unter der Kategorie Datenschutz & Sicherheit, Klicken Sie auf Passwörter. (Wenn unter dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf die Kategorie, um die Liste einzublenden.)
  3. Klicken Sie auf Master-Passwort rücksetzen.
  4. Im folgenden Dialog, wählen Sie Rücksetzen.

Grafiken verwalten

Wenn Sie wollen, können Sie die Anzeige aller Grafiken abschalten. Dies schränkt Ihre Anzeige im Internet sehr ein, ist aber hilfreich, wenn Sie über eine langsame Internet-Verbindung verfügen und die Zeit zum Laden einer Webseite verkürzen wollen.

Sie können auserdem das Wiederholverhalten von Animationen bestimmen, oder Animationen komplett deaktivieren.

Der folgende Abschnitt erklärt wie Sie die Grafik-Einstellungen bedienen. Mit der Standard-Einstellung werden alle Grafiken angezeigt sowie deren Animationen ohne Einschränkungen.

  2. Unter der Kategorie Datenschutz & Sicherheit, Klicken Sie auf Grafiken. (Wenn unter dieser Kategorie keine Einträge angezeigt werden, doppelklicken Sie auf die Kategorie, um die Liste einzublenden.)

Grafik-Anzeigeregeln

Die Grafik-Anzeigeregeln erlauben Ihnen einzustellen, ob &brandShortName; Grafiken anzeigen soll und wenn ja, wo:

  • Keine Grafiken laden: Wählen Sie diese Option, wenn Sie keine Grafiken angezeigt bekommen wollen.
  • Nur Grafiken akzeptieren, die vom Ursprungsserver kommen: Wählen Sie diese Option, wenn Sie Grafiken nur von der gewählten Seite anzeigen möchten.
  • Alle Grafiken akzeptieren: Wählen Sie diese Option, wenn Sie alle Grafiken angezeigt haben wollen. (Diese Option ist die Standardeinstellung.)
  • Keine externen Grafiken in Mail- & Newsgroup-Nachrichten laden: Wenn diese Option gewählt ist, werden keine eingebetteten Grafiken in Mail und Newsgroup-Nachrichten angezeigt. (Diese Checkbox ist standardmässig ausgewählt.)

Wiederholen von animierten Grafiken

Diese Einstellungen bestimmen, wie oft animierte Grafiken ihre Animationen zeigen:

  • So oft, wie in der Grafik angegeben: Wählen Sie diese Option, wenn Sie animierte Grafiken deren Animationen selbst kontrollieren sollen (Diese Option ist standardmässig aktiviert.)
  • Ein Mal: Wählen Sie diese Option, wenn animierte Grafiken deren Animation nur einmal ausführen sollen.
  • Nie: Wählen Sie diese Option, wenn animierte Grafiken deren Animationen nie ausführen sollen.
