# ImloYor — Tajik Dictionary for Firefox

Hunspell spellcheck dictionary for the Tajik language. Adds Tajik (`tg-TJ`) to Firefox's built-in spell checker.

## Features

- 247,000+ words
- Covers modern Tajik vocabulary including official orthographic dictionary
- Supports all Tajik-specific characters: ғ, ӣ, қ, ӯ, ҳ, ҷ

## Installation

Install from [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/imloyor-tajik-dictionary/) or load manually:

1. Download the latest `.xpi` from [Releases](https://github.com/ImloYor/firefox-dictionary/releases)
2. Open Firefox, go to `about:addons`
3. Click the gear icon and select "Install Add-on From File..."

## Usage

1. Right-click any text input field
2. Go to **Languages**
3. Select **Tajik**
4. Misspelled words will be underlined

## Data Sources

- [uniparser-grammar-tajik](https://github.com/timarkh/uniparser-grammar-tajik) (MIT) — morphological dictionary
- [kumitaizabon.tj](http://kumitaizabon.tj/) — Official orthographic dictionary of Tajikistan
