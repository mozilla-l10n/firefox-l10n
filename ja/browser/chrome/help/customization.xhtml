&brandFullName; をカスタマイズする

このセクションでは:
  • ツールバー
  • アドオン(拡張機能とテーマ)

ツールバー

&brandFullName; のツールバーの並べ替え、隠蔽、追加または削除する方法を説明します。 用語に慣れていない方のために説明すると、ツールバーとは Web サイトが表示されている領域の上の (通常は灰色の) 帯のことです。

  • メニューバー: ブラウザメニュー (ファイル、編集、ヘルプなど) を含むツールバーです。
  • ナビゲーションツールバー: ロケーションバーと Web サイトをナビゲートするために使用するボタンを含むツールバーです。
  • ブックマークツールバー: ブックマークツールバーフォルダのブックマークを含むツールバーです。

&brandShortName; のツールバーには、基本的なカスタマイズ方法が 2 通りあります:

  • ツールバーアイテムを並べ替える
  • ツールバーを隠す、追加または削除する

ツールバーアイテムを並べ替える

ツールバー上でアイテム (アイコン、ボタン、テキストボックスなど) を並べ替えるには、 ツールバー上のテキストボックス以外の任意の部分を右クリック&ctrlKey; を押下しながらクリックし、カスタマイズ... を選択してください。ツールバーカスタマイズダイアログを表示します。

並べ替えたいアイテム (メニュー項目を除く) を、ダイアログとツールバー間でドラッグ＆ドロップしてください。 また、すでにツールバー上にあるアイコンも、お望みのところへ並べ替えることができます。 ツールバーやアイテムをあなたの望みどおりに並べ替えたら 完了 をクリックしてください。

特別なツールバーアイテム

ツールバーをカスタマイズする際に利用可能な、特別なアイテムがいくつかあります:

  • 区切り: 小さな垂直線でツールバー上のアイテムを区切ります。
  • スペース: 固定サイズの間隔を空けてツールバー上のアイテムを区切ります。多くのテーマでは、通常 20 ピクセル前後です。
  • 伸縮自在のスペース: スペースは間隔を空けてツールバー上のアイテムを区切ります。 この伸縮自在のスペースは、ツールバー上の空いている間隔をすべて埋めるように自動的にサイズが変更されます。 これは、ツールバーの右側にアイテムを置くのに適しています。

ツールバーを隠す、追加または削除する

画面上に表示したくないときは、ナビゲーションツールバーとブックマークツールバーを隠すことができます。 メニューバーを削除することはできません。メニュー項目(ファイル、編集、ヘルプなど) を異なるツールバーに移動させることができないため、これらは常に画面上に表示されます。

ツールバーを表示または隠すには、表示 > ツールバー を選択し、 表示を切り替えたいツールバー名をクリックしてください。

ツールバーを追加するには、表示 > ツールバー > カスタマイズ を選択してください。 (ダイアログ下部の) ツールバーを追加 をクリックすると、名前を入力するためのダイアログを表示します。 名前を入力して OK をクリックすると、新しいツールバーが現れます。 そこへツールバーアイテムをドラッグ＆ドロップしてください。 あなたのツールバーを削除するには、そこからすべてのツールバーアイテムをドラッグ＆ドロップして取り除き、 完了 をクリックしてください。また、あなたのツールバーを隠す方法については前述の通りです。

アドオン(拡張機能とテーマ)

アドオンは、&brandShortName; の機能や外観を追加、変更するための小さなソフトウェアの部品です。 アドオンには拡張機能とテーマの 2 種類があります。

拡張機能
拡張機能は &brandShortName; に新しい機能を追加します。 ツールバーボタンから完全に新しい機能まで、何でも追加することができます。 拡張機能は最小限のサイズと外観で、アプリケーションを個人的な要望に合わせてカスタマイズします。
テーマ
テーマは &brandShortName; の外観を変更します。 &brandShortName; の外観と操作感をあなたのお好みに変えることができます。 テーマは単純にボタンの画像を変更するだけでなく、すべての &brandShortName; の部品の外観を変更することができます。

アドオンを取得しインストールする

&brandShortName; Add-ons から多くのアドオンをダウンロードしてインストールすることができます。 インターネット検索で、さらに多くのアドオンを見つけることができるでしょう。

アドオンをインストールするためのリンクをクリックすると、 インストールの許可を求めるダイアログが表示されます。 ダウンロードとインストールを許可するか、または手続きの中止を選んでください。

アドオンマネージャを使用する

アドオンはアドオンマネージャで管理されます。アドオンマネージャを開くには、 ツール > アドオン を選択します。

アドオンを更新する

既定では、インストールされたアドオンの新しいバージョンが利用可能かどうか、 &brandShortName; が定期的に確認します。利用可能なときは、次回の起動時に最新版をインストールします。 この動作は&pref.pluralCaps;の 詳細 パネルの 更新 タブ から変更することができます。

更新を手動で確認するには、拡張機能 または テーマ パネルを選択して 更新を確認 ボタンをクリックしてください。最新版が利用可能なときは、 インストールしたいアドオンを選ぶための 更新 パネルが表示されます。 アドオンを更新するには、更新をインストール ボタンをクリックしてください。 変更を反映するには、&brandShortName; を再起動する必要があります。

アドオンを無効化、有効化、削除する

拡張機能の使用を止めたいときは一時的に無効化することもできます。 あとで再び有効にするか削除するかを選べるため、こちらのほうが簡単です。 テーマは現在のもの以外はすべて自動的に無効化されるため、削除のみ可能です。

拡張機能を無効または有効にするには、拡張機能を選択し、無効 または 有効 ボタンをクリックしてください。 &brandShortName; からアドオンを削除するには、削除したいアドオンを選択して 削除 ボタンをクリックしてください。 すべての変更を反映するには、&brandShortName; を再起動する必要があります。

さらなる機能

サイトを表示 と (アドオン) について などの機能は、 アドオンを選択して &ctrlKey; を押下しながらクリック右クリックし、コンテキストメニューから選択します。

拡張機能またはテーマのための操作

拡張機能を設定する

拡張機能は多くの場合、その機能を設定するためのダイアログを提供しています。 拡張機能の&pref.plural;ダイアログを開くには、拡張機能パネルを選択し、拡張機能を選択して 設定 ボタンをクリックするか、拡張機能をダブルクリックしてください。

テーマを切り替える

インストールされたテーマを切り替えるには、テーマパネルを選択し、テーマを選択して テーマを使用 ボタンをクリックしてください。 変更を反映するには、&brandShortName; を再起動する必要があります。
