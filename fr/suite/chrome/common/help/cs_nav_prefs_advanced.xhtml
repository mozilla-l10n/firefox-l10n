Préférences avancées - &pref.advanced.title;

Cette section décrit l'utilisation du panneau principal des préférences avancées. Si vous ne l'utilisez pas en ce moment, suivez ces étapes :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et choisissez &preferencesCmd.label;.
  2. Cliquez sur la catégorie &pref.advanced.title;.

La fenêtre principale des préférences avancées vous permet de choisir différentes options pour Java, les mots de passes FTP et le démarrage rapide de &brandShortName; :

&enbJavaCheck.label; :
Sélectionnez cette option pour permettre à &brandShortName; d'interpréter les pages contenant du Java. Le Java est utilisé sur certaines pages Web pour incorporer du contenu dynamique et des objets multimédia.
&systemPref.caption; :
Sélectionnez cette option pour utiliser les préférences de votre système. Elles auront ainsi la priorité sur celles de &brandShortName;.

Préférences avancées - &pref.scripts.title;

Cette section décrit l'utilisation du panneau &pref.scripts.title; des préférences avancées. Si vous ne l'utilisez pas en ce moment, suivez ces étapes :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et choisissez &preferencesCmd.label;.
  2. Sous la catégorie &pref.advanced.title;, cliquez sur&pref.scripts.title;. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie &pref.advanced.title; pour développer la liste.

Le panneau des préférences &pref.scripts.title; vous permet de contrôler l'utilisation du JavaScript et des plugins dans &brandShortName; :

&enableJavaScript.label; :
Cochez ces cases pour activer le Javascript :
&navigator.label; :
Active le JavaScript pour les pages s'ouvrant dans le navigateur.
&enbPluginCheckMailNews.label; :
Active le JavaScript pour le courrier électronique ou les messages des groupes de discussions.
&allowScripts.label;
Cochez ces cases pour contrôler l'utilisation du JavaScript :
&allowWindowMoveResize.label; :
Permet aux fenêtres ouvertes d'être redimensionnées ou déplacées.
&allowWindowFlip.label; :
Permet aux fenêtres d'être placées aux-dessus ou en-dessous des autres fenêtres.
&allowHideStatusBar.label; :
Autorise le JavaScript à cacher la barre d'état.
&allowWindowStatusChange.label; :
Permet de changer le texte de la barre d'état, comme par exemple un texte défilant.
&allowWindowImageSrcChange.label; :
Permet le changement ou l'animation d'images, comme dans les effets de survol (des images qui changent quand le curseur de la souris est placé au-dessus d'elles).
&allowContextmenuDisable.label; :
Pour certaines raisons, notamment de droit d'auteur, certaines pages Web utilisent des scripts bloquant le menu contextuel. Décochez cette case si vous désirez ne pas autoriser ces scripts à bloquer le menu contextuel du navigateur.
&enablePlugin.label; :
&enbPluginCheckMailNews.label; :
Cette option vous permet d'activer les plugins pour les fenêtres du courrier et des groupes.

Pour plus d'informations sur les plugins, voir la section Plugins et téléchargements.

Préférences avancées - &pref.cache.title;

Cette section décrit l'utilisation du panneau &pref.cache.title; des préférences avancées. Si vous ne l'utilisez pas en ce moment, suivez ces étapes :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et choisissez &preferencesCmd.label;.
  2. Sous la catégorie &pref.advanced.title;, cliquez sur &pref.cache.title;. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie &pref.advanced.title; pour étendre la liste.

Le panneau &pref.cache.title; vous permet d'ajuster le cache mémoire et le cache disque de &brandShortName; :

&pref.cache.caption; :
Le cache est un espace disque sur lequel sont enregistrées les pages que vous visitez le plus fréquemment. Le cache augmente la vitesse d'affichage des pages en permettant au navigateur de récupérer les pages localement plutôt que sur le réseau. Plus l'espace alloué au cache est important, plus la quantité de fichiers stockées sera grande.
&diskCache.label; __ &mbytes; :
Saisissez la taille du cache disque que vous désirez allouer à &brandShortName;. Le cache disque est enregistré sur votre disque dur et peut encore être utilisé, même après avoir éteint votre ordinateur.
&clearDiskCache.label; :
Cliquez sur ce bouton si vous désirez effacer tous les fichiers contenus dans le cache.
&diskCacheFolder.label; :
Indique le chemin du répertoire où &brandShortName; enregistre votre cache disque.
&chooseDiskCacheFolder.label; :
Cliquez sur ce bouton si vous voulez choisir le dossier du cache disque.
&docCache;
&everyTimeRadio.label; :
Sélectionnez cette option si vous désirez que &brandShortName; compare la page Web du serveur avec celle du cache chaque fois que vous la visitez.
&autoRadio.label; :
Sélectionnez cette option si vous désirez que &brandShortName; compare la page Web du serveur avec celle du cache lorsque le serveur indique que la page a expiré.
&oncePsessionRadio.label; :
Sélectionnez cette option si vous désirez que &brandShortName; compare la page Web du serveur avec celle du cache après chaque redémarrage de &brandShortName;.
&neverRadio.label; :
Sélectionnez cette option si vous ne désirez pas que &brandShortName; compare les pages du cache avec celles du réseau.
&prefetchTitle.label; :
Certaines pages contiennent des liens définis pour être préchargées pendant les pauses du navigateur. Cochez cette case pour activer cette fonction qui améliore la vitesse d'affichage.

Le panneau &pref.proxies.title; vous permet de configurer &brandShortName; pour utiliser un proxy :

&directTypeRadio.label; :
Choisissez cette option si vous ne désirez pas utiliser un proxy.
&wpadTypeRadio.label; :

&manualTypeRadio.label; :
Choisissez cette option si vous ne possédez pas l'adresse de votre proxy (URL), ou si la découverte automatique du proxy n'a pas correctement paramétré celui-ci.
Proxies HTTP, SSL, FTP ou Gopher :
Saisissez le nom ou l'URL du serveur proxy ou son IP numérique.
&port.label;
Saisissez le numéro du port dans le champ Port. Cliquez sur &advanced.label; pour paramétrer les &pref.proxies.advanced.title;.
Pas de proxy pour :
Saisissez les noms de domaine ou les adresses IP pour lesquels vous ne désirez pas utilisez un proxy. Séparez chaque nom de domaine par une virgule (exemple : .mozilla.com, .mozilla-europe.org, 192.168.1.0/24).

Préférences avancées du proxy

Si vous voulez utiliser différents proxy pour différents protocoles, ou si vous avez besoin d'utiliser un proxy SOCKS :

Proxies HTTP, SSL, FTP ou Gopher :
Saisissez le nom ou l'adresse IP du serveur de proxy. Saisissez le numéro du port dans le champ Port.

Remarque : Si vous utilisez les mêmes paramètres pour tous les protocoles, cliquez sur &reuseProxy.label;.

&socks.label;

&socks4.label;, &socks5.label; :
Lorsque vous renseignez un proxy SOCKS, sélectionnez &socks4.label; ou &socks5.label;, selon la version du SOCKS utilisée pour le proxy.
&socksRemoteDNS.label; :
Sélectionnez cette option pour utiliser le proxy SOKS pour la résolution des noms de serveurs. Cette option est recommandée pour les proxy SOKS v5.

Préférences avancées - &pref.http.title;

Cette partie décrit comment utiliser le panneau &pref.http.title; des préférences avancées. Si vous ne l'utilisez pas en ce moment, suivez ces étapes :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et choisissez &preferencesCmd.label;.
  2. Sous la catégorie &pref.advanced.title;, cliquez sur &pref.http.title;. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie &pref.advanced.title; pour développer la liste.

Ce panneau vous permet de configurer le protocole HTTP :

&prefDirect.label; - &prefProxy.label; :
Choisissez la version du HTTP ainsi que les options de connexions directe et par proxy.
&prefEnableHTTP10.label; :
Choisissez cette option pour utiliser la version originale du HTTP, standardisée en 1996.
&prefEnableHTTP11.label; :
Choisissez cette option pour utiliser la nouvelle version du HTTP, qui offre des performances améliorées, comme l'utilisation plus efficace des connexions HTTP, un meilleur support pour la mise en cache du côté client, des requêtes HTTP multiples (pipelining), et un contrôle plus fin sur l'expiration et les méthodes de remplacement du cache.
&prefEnableKeepAlive.label; :
Sélectionnez cette option pour garder une connexion ouverte afin de faire des requêtes HTTP supplémentaires, augmentant la vitesse.
&prefEnablePipelining.label; :
Sélectionnez cette option pour permettre le pipelining, lequel permet d'envoyer immédiatement plus d'une requête HTTP au serveur, réduisant les délais de chargement des pages Web.

Remarque : Le pipelining est disponible seulement avec HTTP 1.1.

Préférences avancées - &mousewheel.label;

Cette section décrit l'utilisation du panneau &pref.mouseWheel.title; des préférences avancées. Certaines souris récents possèdent deux molettes, ou un bouton qui permet de modifier la direction de défilement. Le comportement vertical de la molette est paramétrable dans le panneau du haut &mousewheelGroup.label; alors que le comportement horizontal de la molette est paramétrable dans le panneau du bas &mousewheelHorizGroup.label;. Si vous ne l'utilisez pas en ce moment, suivez ces étapes :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et choisissez &preferencesCmd.label;.
  2. Sous la catégorie &pref.advanced.title;, cliquez sur &mousewheel.label;. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie &pref.advanced.title; pour développer la liste.

Le panneau &pref.mouseWheel.title; vous permet de contrôler l'utilisation de la molette de la souris dans &brandShortName;.

Utilisez les onglets pour paramétrer les actions qui correspondent à chaque touche de modification. Les actions possibles, communes à toutes les touches, sont les suivantes :

&scroll.label; __ &scrollLines.label; :
Choisissez cette option pour faire défiler le document du nombre de lignes (ou de caractères pour les mouvements horizontaux) entré dans ce champ.
&useSystemDefault.label;e :
Choisissez cette option pour ignorer le nombre de lignes (ou de caractères) entré et utiliser plutôt les paramètres par défaut du système.
&scrollPgUpPgDn.label; - &scrollPgLtPgRt.label; :
Choisissez cette option pour faire défiler d'une page vers le haut ou vers le bas. Cette option permet une plus grande rapidité dans la navigation, mais elle est moins précise que le défilement d'un nombre défini de lignes.
&history.label; :
Choisissez cette option pour naviguer dans les pages visitées, stockées dans l'historique.
&textsize.label; :
Choisissez cette option pour utiliser la molette de la souris pour augmenter ou diminuer la taille du texte sur une page Web. Ce réglage peut vous aider à lire une page avec plus de confort, en diminuant ou en augmentant la taille du texte.

Remarque : Si votre souris ne possède pas de mode de défilement horizontal, les paramètres du panneau &mousewheelHorizGroup.label; seront ignorés.

Préférences avancées - &Inspector.label;

Cette section décrit l'utilisation du panneau Inspecteur DOM des préférences avancées. Si vous ne l'utilisez pas en ce moment, suivez ces étapes :

Le panneau &Inspector.label; vous permet de configurer l'affichage de l'inspecteur DOM, présent dans &brandShortName; :

&elementBlinking.label; :
Lorsque vous sélectionnez un élément de la page Web que vous examinez, vous avez la possibilité de le faire clignoter pour vous aider à mieux le repérer. Les options suivantes vous permettent de paramétrer ce clignotement
&blinkSelectedElement.label; :
Cochez cette case si vous voulez faire clignoter l'élément sélectionné.
&borderColor.label; :
Cliquez sur la case de couleur et choisissez la couleur que vous désirez donner à la bordure clignotante.
&borderWidth.label; :
Indiquez, en pixel, la largeur de la bordure de l'élément sélectionné.
&blinkDuration.label; :
Indiquez, en millième de seconde, le temps de clignotement de l'élément sélectionné.
&blinkSpeed.label; :
Indiquez, en millième de seconde, la vitesse du clignotement. Plus le nombre que vous saisissez est petit, plus le clignotement est rapide.
&invertColors.label; :
Cochez cette case pour faire clignoter l'élément sélectionné en inversant les couleurs définies dans la page.
&Sidebar.label; :
Vous avez la possibilité d'installer le panneau de l'inspecteur DOM dans la barre latérale. Vous pouvez ainsi examiner la page en cours d'affichage sans ouvrir l'inspecteur DOM lui-même.
&install.label; :
Cliquez sur ce bouton pour installer le panneau de l'inspecteur DOM dans la barre latérale.

Pour plus d'informations sur l'inspecteur DOM, voir la page Web Inspecteur DOM.
