Nástroje pro vývoj webových aplikací

Pro webové vývojáře &brandShortName; poskytuje několik pomocných nástrojů pro vývoj a ladění webových aplikací. Některé z nich jsou součástí volitelné instalace. Tyto nástroje naleznete v nabídce Nástroje pod položkou Vývoj webových stránek.

  • Chybová konzola: okno konzole, kde naleznete informace o chybách souvisejících s JavaScriptem a CSS ve webových aplikacích či přímo v &brandShortName;. Ve výchozím nastavení jsou zobrazovány chyby v CSS a JavaScriptu. Konzole též může být využita k zobrazení logovaných zpráv ze XULu a JavaScriptu.
  • DOM Inspektor: nástroj, který může být použit k prohlížení a editaci DOM v jakémkoliv webovém dokumentu či XUL aplikaci.

    On-line dokumenty:

    • Introduction to the DOM Inspector (Ian Oeschger) [Eng.]
    • DOM Inspector FAQ (mozilla.org) [Eng.]
    • Creating Applications with Mozilla - Appendix B3: The DOM Inspector [Eng.]
  • JavaScript Debugger: též známý jako Venkman. Tento nástroj umožňuje ladit JavaScript v balíku &brandShortName;. Podporuje krokování, zarážky při ladění a řadu dalších možností.

    On-line dokumenty:

    • Venkman JavaScript Debugger Homepage [Eng.]
    • Learning the JavaScript debugger Venkman (Svend Tofte) [Eng.]
    • Debugging JavaScript Using Venkman (webreference.com, article series) [Eng.]
    • Venkman Development Homepage [Eng.]
